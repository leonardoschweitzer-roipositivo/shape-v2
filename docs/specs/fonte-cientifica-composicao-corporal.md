## 1. VISÃO GERAL

### 1.1 O que é esta SPEC?

Esta SPEC contém o conhecimento científico sobre **composição corporal** que o VITRÚVIO deve usar como base para:

- Interpretar medições de gordura corporal
- Classificar atletas por faixas de % de gordura
- Calcular e interpretar FFMI (Fat-Free Mass Index)
- Comparar diferentes métodos de medição
- Estabelecer metas realistas de composição corporal

### 1.2 Definição

> **Composição Corporal** é a análise dos diferentes componentes que formam o corpo humano, principalmente massa gorda, massa magra (muscular) e massa óssea.

```
MODELO DE DOIS COMPARTIMENTOS:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  PESO CORPORAL TOTAL                                                   │
│         │                                                               │
│         ├──── MASSA GORDA (Fat Mass - FM)                              │
│         │     • Gordura essencial (órgãos, cérebro, medula)            │
│         │     • Gordura de armazenamento (subcutânea, visceral)        │
│         │                                                               │
│         └──── MASSA LIVRE DE GORDURA (Fat-Free Mass - FFM)             │
│               • Músculos                                                │
│               • Ossos                                                   │
│               • Órgãos                                                  │
│               • Água corporal                                           │
│               • Tecido conjuntivo                                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 DEXA (Dual-Energy X-ray Absorptiometry)

```
DEXA - PADRÃO-OURO:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  O QUE MEDE:                                                           │
│  • Massa gorda total e regional                                        │
│  • Massa magra total e regional                                        │
│  • Densidade mineral óssea                                             │
│  • Gordura visceral (VAT)                                              │
│  • Relação androide/ginoide                                            │
│                                                                         │
│  PRECISÃO:                                                             │
│  • CV (coeficiente de variação): ~2% para % gordura                    │
│  • CV: ~0.5% para massa magra                                          │
│  • Precisão: ±1-2 pontos percentuais                                   │
│                                                                         │
│  VANTAGENS:                                                            │
│  • Análise regional (braços, pernas, tronco)                           │
│  • Detecta assimetrias musculares                                      │
│  • Mede gordura visceral                                               │
│  • Não afetado por hidratação                                          │
│  • Rápido (~10 minutos)                                                │
│                                                                         │
│  LIMITAÇÕES:                                                           │
│  • Custo elevado                                                       │
│  • Requer clínica/laboratório                                          │
│  • Exposição mínima a radiação                                         │
│  • Diferentes máquinas podem dar resultados diferentes                 │
│                                                                         │
│  NOTA IMPORTANTE:                                                      │
│  DEXA tende a reportar % de gordura ~3-5% MAIOR que outros métodos    │
│  porque detecta gordura intramuscular e em órgãos.                     │
│  Um "15%" no DEXA pode ser "11-12%" em dobras cutâneas.               │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 4. FFMI (FAT-FREE MASS INDEX)

### 4.1 Definição e Cálculo

```
FFMI - ÍNDICE DE MASSA LIVRE DE GORDURA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  DEFINIÇÃO:                                                            │
│  Medida que avalia a quantidade de massa muscular relativa à altura.   │
│  É como o IMC, mas apenas para massa magra.                            │
│                                                                         │
│  FÓRMULA BÁSICA:                                                       │
│  ───────────────                                                       │
│  FFMI = (Massa Magra em kg) / (Altura em metros)²                      │
│                                                                         │
│  Onde:                                                                 │
│  Massa Magra = Peso × (1 - % Gordura / 100)                            │
│                                                                         │
│  FÓRMULA NORMALIZADA (Kouri et al., 1995):                             │
│  ─────────────────────────────────────────────                         │
│  FFMI Normalizado = FFMI + 6.3 × (1.80 - Altura em metros)             │
│                                                                         │
│  A normalização ajusta para altura de 1.80m (5'11"),                   │
│  permitindo comparação justa entre pessoas de diferentes alturas.      │
│                                                                         │
│  EXEMPLO:                                                              │
│  ──────────                                                            │
│  Peso: 82.5 kg                                                         │
│  Altura: 1.73 m                                                        │
│  % Gordura: 15.8%                                                      │
│                                                                         │
│  Massa Magra = 82.5 × (1 - 0.158) = 69.46 kg                           │
│  FFMI = 69.46 / 1.73² = 23.2                                           │
│  FFMI Normalizado = 23.2 + 6.3 × (1.80 - 1.73) = 23.6                  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 5.1 IMC (Índice de Massa Corporal)

```
IMC - LIMITAÇÕES PARA ATLETAS:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  FÓRMULA:                                                              │
│  IMC = Peso (kg) / Altura (m)²                                         │
│                                                                         │
│  CLASSIFICAÇÃO OMS:                                                    │
│  • < 18.5: Abaixo do peso                                              │
│  • 18.5-24.9: Peso normal                                              │
│  • 25-29.9: Sobrepeso                                                  │
│  • ≥ 30: Obesidade                                                     │
│                                                                         │
│  PROBLEMA PARA ATLETAS:                                                │
│  ─────────────────────                                                 │
│  IMC não diferencia músculo de gordura!                                │
│                                                                         │
│  Exemplo:                                                              │
│  • Atleta: 90 kg, 1.80 m, 12% gordura                                  │
│  • IMC = 27.8 → "Sobrepeso" (FALSO!)                                   │
│                                                                         │
│  • Sedentário: 80 kg, 1.80 m, 28% gordura                              │
│  • IMC = 24.7 → "Normal" (mas tem muita gordura!)                      │
│                                                                         │
│  CONCLUSÃO:                                                            │
│  IMC é útil para população geral, mas INÚTIL para atletas.             │
│  Use % de gordura e FFMI em vez de IMC.                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 6. COMO O VITRÚVIO DEVE USAR

### 6.1 Algoritmo de Avaliação

```typescript
interface AvaliacaoComposicao {
  peso: number              // kg
  altura: number            // cm
  gorduraCorporal: number   // %
  metodoMedicao: 'DEXA' | 'DOBRAS' | 'BIA' | 'FITA' | 'VISUAL'
  idade: number
  sexo: 'M' | 'F'
  circunferenciaAbdominal?: number  // cm
}

interface ResultadoComposicao {
  massaMagra: number          // kg
  massaGorda: number          // kg
  ffmi: number
  ffmiNormalizado: number
  classificacaoGordura: string
  classificacaoFFMI: string
  imc: number
  recomendacoes: string[]
}

function avaliarComposicao(dados: AvaliacaoComposicao): ResultadoComposicao {
  const alturaM = dados.altura / 100
  
  // Calcular massas
  const massaGorda = dados.peso * (dados.gorduraCorporal / 100)
  const massaMagra = dados.peso - massaGorda
  
  // Calcular FFMI
  const ffmi = massaMagra / (alturaM * alturaM)
  const ffmiNormalizado = ffmi + 6.3 * (1.80 - alturaM)
  
  // IMC
  const imc = dados.peso / (alturaM * alturaM)
  
  // Classificar % gordura
  let classificacaoGordura: string
  if (dados.sexo === 'M') {
    if (dados.gorduraCorporal < 6) classificacaoGordura = 'Gordura essencial (muito baixo)'
    else if (dados.gorduraCorporal < 10) classificacaoGordura = 'Atleta de elite'
    else if (dados.gorduraCorporal < 14) classificacaoGordura = 'Atleta'
    else if (dados.gorduraCorporal < 18) classificacaoGordura = 'Fitness'
    else if (dados.gorduraCorporal < 25) classificacaoGordura = 'Aceitável'
    else if (dados.gorduraCorporal < 30) classificacaoGordura = 'Sobrepeso'
    else classificacaoGordura = 'Obeso'
  } else {
    if (dados.gorduraCorporal < 14) classificacaoGordura = 'Gordura essencial (muito baixo)'
    else if (dados.gorduraCorporal < 18) classificacaoGordura = 'Atleta de elite'
    else if (dados.gorduraCorporal < 23) classificacaoGordura = 'Atleta'
    else if (dados.gorduraCorporal < 28) classificacaoGordura = 'Fitness'
    else if (dados.gorduraCorporal < 33) classificacaoGordura = 'Aceitável'
    else if (dados.gorduraCorporal < 39) classificacaoGordura = 'Sobrepeso'
    else classificacaoGordura = 'Obesa'
  }
  
  // Classificar FFMI
  let classificacaoFFMI: string
  if (dados.sexo === 'M') {
    if (ffmiNormalizado < 18) classificacaoFFMI = 'Abaixo da média'
    else if (ffmiNormalizado < 20) classificacaoFFMI = 'Média'
    else if (ffmiNormalizado < 22) classificacaoFFMI = 'Acima da média'
    else if (ffmiNormalizado < 23) classificacaoFFMI = 'Avançado'
    else if (ffmiNormalizado < 25) classificacaoFFMI = 'Elite natural'
    else if (ffmiNormalizado < 26) classificacaoFFMI = 'Excepcional'
    else classificacaoFFMI = 'Além do limite natural típico'
  } else {
    if (ffmiNormalizado < 15) classificacaoFFMI = 'Abaixo da média'
    else if (ffmiNormalizado < 17) classificacaoFFMI = 'Média'
    else if (ffmiNormalizado < 18) classificacaoFFMI = 'Acima da média'
    else if (ffmiNormalizado < 19) classificacaoFFMI = 'Avançada'
    else if (ffmiNormalizado < 21) classificacaoFFMI = 'Elite natural'
    else classificacaoFFMI = 'Excepcional'
  }
  
  // Recomendações
  const recomendacoes: string[] = []
  
  // Ajustar expectativa por método de medição
  if (dados.metodoMedicao === 'DEXA') {
    recomendacoes.push('Medição DEXA: valores ~3-5% mais altos que outros métodos')
  } else if (dados.metodoMedicao === 'BIA') {
    recomendacoes.push('Bioimpedância: sensível à hidratação, usar para tendências')
  }
  
  // Recomendações por classificação
  if (classificacaoGordura.includes('Obeso') || classificacaoGordura.includes('Sobrepeso')) {
    recomendacoes.push('Foco em déficit calórico moderado para reduzir gordura')
  }
  
  if (ffmiNormalizado < (dados.sexo === 'M' ? 20 : 17)) {
    recomendacoes.push('Potencial significativo para ganho de massa muscular')
  }
  
  if (dados.circunferenciaAbdominal) {
    const limiteRisco = dados.sexo === 'M' ? 102 : 88
    if (dados.circunferenciaAbdominal > limiteRisco) {
      recomendacoes.push('Circunferência abdominal elevada - priorizar redução de gordura visceral')
    }
  }
  
  return {
    massaMagra: Math.round(massaMagra * 10) / 10,
    massaGorda: Math.round(massaGorda * 10) / 10,
    ffmi: Math.round(ffmi * 10) / 10,
    ffmiNormalizado: Math.round(ffmiNormalizado * 10) / 10,
    classificacaoGordura,
    classificacaoFFMI,
    imc: Math.round(imc * 10) / 10,
    recomendacoes
  }
}
```

### 6.2 Exemplo de Aplicação

```
ATLETA: Leonardo
─────────────────────────────────────────────────────
Peso: 82.5 kg
Altura: 173 cm
Gordura corporal: 15.8%
Método: Dobras cutâneas
Idade: 32 anos
Sexo: Masculino
Circunferência abdominal: 82 cm

CÁLCULOS:
─────────────────────────────────────────────────────

Massa gorda = 82.5 × 0.158 = 13.0 kg
Massa magra = 82.5 - 13.0 = 69.5 kg

FFMI = 69.5 / 1.73² = 23.2
FFMI Normalizado = 23.2 + 6.3 × (1.80 - 1.73) = 23.6

IMC = 82.5 / 1.73² = 27.6

RESULTADOS:
─────────────────────────────────────────────────────
• Massa magra: 69.5 kg
• Massa gorda: 13.0 kg
• % Gordura: 15.8% → Classificação: "Fitness"
• FFMI Normalizado: 23.6 → Classificação: "Elite Natural"
• IMC: 27.6 → (ignorar, atleta muscular)
• Circunferência abdominal: 82 cm → Normal (< 94 cm)

ANÁLISE:
─────────────────────────────────────────────────────
• Excelente desenvolvimento muscular (top 5% naturais)
• % gordura em faixa saudável e estética
• Sem risco de gordura visceral elevada
• Potencial para melhorar composição:
  - Reduzir para 12% BF → FFMI ~23.8-24
  - Ou manter 15-16% e focar em hipertrofia
```

### 7.2 Regras de Ouro

```
REGRAS DO VITRÚVIO PARA COMPOSIÇÃO CORPORAL:

1. % GORDURA > IMC
   • Sempre usar % gordura, não IMC, para atletas
   • IMC classifica erroneamente pessoas musculosas

2. CONSISTÊNCIA > PRECISÃO
   • Usar mesmo método, mesmo avaliador, mesmo horário
   • Tendências são mais importantes que valores absolutos

3. DEXA REPORTA MAIS ALTO
   • DEXA mede gordura intramuscular e em órgãos
   • Um "15% DEXA" equivale a ~11-12% em dobras

4. FFMI PARA AVALIAR DESENVOLVIMENTO
   • Homens: meta de FFMI 22-24 é excelente
   • Mulheres: meta de FFMI 18-20 é excelente
   • FFMI > 25 (homens) ou > 21 (mulheres) é raro naturalmente

5. NÃO EXISTE % GORDURA "IDEAL" UNIVERSAL
   • Depende do objetivo (saúde, estética, performance)
   • Depende da sustentabilidade
   • Muito baixo é tão problemático quanto muito alto

6. GORDURA VISCERAL É O VERDADEIRO PERIGO
   • Circunferência abdominal > 102 cm (H) ou > 88 cm (M) = risco
   • Priorizar redução de gordura visceral sobre subcutânea
```
