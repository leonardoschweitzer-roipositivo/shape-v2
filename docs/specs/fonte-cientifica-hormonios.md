## 1. VISÃO GERAL

### 1.1 O que é esta SPEC?

Esta SPEC contém o conhecimento científico sobre **hormônios relevantes para hipertrofia e composição corporal** que o VITRÚVIO deve usar para:

- Entender o papel dos hormônios nos ganhos musculares
- Explicar diferenças entre homens e mulheres
- Orientar sobre fatores que afetam hormônios (sono, dieta, treino)
- Contextualizar o envelhecimento e seus efeitos
- Desmistificar conceitos errados sobre "hormônios pós-treino"

### 1.3 O Grande Debate: Hormônios Agudos vs Crônicos

**DESCOBERTA IMPORTANTE (Phillips et al., West et al., Morton et al.):**

> "As elevações agudas pós-exercício de hormônios anabólicos (testosterona, GH, IGF-1) **NÃO** são preditoras de hipertrofia."

```
O QUE REALMENTE IMPORTA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  ❌ NÃO PREDIZ HIPERTROFIA:                                            │
│  • Pico de testosterona pós-treino                                     │
│  • Pico de GH pós-treino                                               │
│  • Pico de IGF-1 pós-treino                                            │
│                                                                         │
│  ✅ PREDIZ HIPERTROFIA:                                                │
│  • Conteúdo de receptores androgênicos no músculo                      │
│  • Sinalização intracelular (mTOR, p70S6K)                             │
│  • Níveis basais adequados de hormônios                                │
│  • Tensão mecânica + estresse metabólico                               │
│                                                                         │
│  CONCLUSÃO:                                                            │
│  Hormônios funcionam como "permissivos" - precisam estar               │

│                                                                         │
```

## 2. TESTOSTERONA

### 2.1 O que é e Como Funciona

A testosterona é o principal hormônio androgênico-anabólico:

**Efeitos Anabólicos:**
- Estimula síntese proteica muscular
- Inibe degradação proteica (anti-catabólico)
- Promove hipertrofia de fibras tipo I e tipo II
- Aumenta número de células satélite
- Ativa via Akt/mTOR/S6K1

**Mecanismo:**
1. Testosterona liga-se ao Receptor Androgênico (AR)
2. Complexo T-AR transloca para o núcleo
3. Liga-se a elementos de resposta androgênica (ARE)
4. Ativa transcrição de genes anabólicos
5. Também ativa IGF-1 local no músculo

### 2.2 Níveis Normais

| Grupo | Testosterona Total | Testosterona Livre |
|-------|-------------------|-------------------|
| **Homens (20-30 anos)** | 300-1000 ng/dL | 9-30 ng/dL |
| **Homens (40-50 anos)** | 250-850 ng/dL | 6-25 ng/dL |
| **Homens (60+ anos)** | 200-700 ng/dL | 4-20 ng/dL |
| **Mulheres (20-40 anos)** | 15-70 ng/dL | 0.3-1.9 ng/dL |
| **Mulheres (pós-menopausa)** | 10-40 ng/dL | 0.1-0.8 ng/dL |

**Observação:** Mulheres têm 10-20x menos testosterona total e ~200x menos testosterona livre que homens.

### 2.3 Declínio com a Idade

```
DECLÍNIO DA TESTOSTERONA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  HOMENS:                                                               │
│  • A partir dos 30-35 anos: queda de 1-2% ao ano                       │
│  • Aos 50 anos: ~15-30% menor que aos 25                               │
│  • Aos 70 anos: ~40-50% menor que aos 25                               │
│  • "Andropausa" é gradual (não abrupta como menopausa)                 │
│                                                                         │
│  MULHERES:                                                             │
│  • Declínio gradual até menopausa                                      │
│  • Menopausa: queda drástica de estrogênio                             │
│  • Testosterona também cai significativamente                          │
│                                                                         │
│  IMPLICAÇÕES PRÁTICAS:                                                 │
│  ─────────────────────                                                 │
│  • Ganhos musculares mais lentos com idade                             │
│  • Recuperação mais demorada                                           │
│  • Maior importância de sono, nutrição, descanso                       │
│  • Treino ainda funciona - apenas mais lento                           │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 2.4 Fatores que Afetam Testosterona

**Aumentam (naturalmente):**
| Fator | Impacto | Evidência |
|-------|---------|-----------|
| Sono 7-9h | +10-15% vs sono ruim | Forte |
| Treino de força | Mantém níveis saudáveis | Forte |
| Gordura corporal saudável | Otimiza produção | Forte |
| Vitamina D adequada | Corrige deficiência | Moderada |
| Zinco adequado | Corrige deficiência | Moderada |
| Reduzir estresse crônico | Reduz cortisol | Moderada |

**Diminuem:**
| Fator | Impacto | Evidência |
|-------|---------|-----------|
| Sono <6h | -10-15% ou mais | Forte |
| Déficit calórico severo | -20-40% | Forte |
| Obesidade | -30-50% | Forte |
| Estresse crônico | Variável | Moderada |
| Álcool em excesso | -20-30% | Moderada |
| Overtraining | Variável | Moderada |

### 2.5 Receptor Androgênico (AR) - O Verdadeiro Preditor

```
RECEPTOR ANDROGÊNICO E HIPERTROFIA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  ESTUDO MORTON ET AL. (2018):                                          │
│  ─────────────────────────────                                         │
│  • 49 homens treinados, 12 semanas de treino                           │
│  • Mediram hormônios circulantes e intramusculares                     │
│  • Mediram conteúdo de AR no músculo                                   │
│                                                                         │
│  RESULTADO:                                                            │
│  • Hormônios circulantes: SEM correlação com hipertrofia               │
│  • Hormônios intramusculares: SEM correlação                           │
│  • Conteúdo de AR: CORRELAÇÃO SIGNIFICATIVA (p<0.01)                   │
│                                                                         │
│  CONCLUSÃO:                                                            │
│  "É a quantidade de receptores no músculo, não os hormônios            │
│  circulantes, que determina a resposta hipertrófica."                  │
│                                                                         │
│  IMPLICAÇÃO:                                                           │
│  O treino de força AUMENTA conteúdo de AR (upregulation)               │
│  por 1-2 dias após a sessão, potencializando efeitos da T.             │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

> "Homens e mulheres têm ganhos RELATIVOS similares em massa muscular e força após treino de força."

```
│  • Homens podem ganhar mais kg absolutos de músculo                    │
```

## 3. CORTISOL

### 3.1 O que é e Como Funciona

Cortisol é o principal glicocorticoide (hormônio do estresse):

**Funções Fisiológicas (necessárias):**
- Mobiliza glicose para energia
- Mobiliza aminoácidos e ácidos graxos
- Regula resposta inflamatória
- Mantém pressão arterial
- Ritmo circadiano (pico matinal)

**Efeitos Catabólicos (quando crônico/excessivo):**
- Aumenta degradação proteica muscular
- Ativa sistema ubiquitina-proteassoma
- Reduz síntese proteica
- Atrofia principalmente fibras tipo II
- Antagoniza efeitos da testosterona

### 3.2 Cortisol NÃO é o Vilão

```
DESMISTIFICANDO O CORTISOL:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  ❌ MITO: "Cortisol é sempre ruim para ganhos"                         │
│                                                                         │
│  ✅ REALIDADE:                                                         │
│  ─────────────                                                         │
│  • Cortisol AGUDO pós-treino é NORMAL e NECESSÁRIO                     │
│  • Faz parte do processo de adaptação                                  │
│  • Ajuda na resposta inflamatória para remodelação                     │
│  • Mobiliza energia durante o treino                                   │
│                                                                         │
│  ⚠️ PROBLEMA REAL:                                                     │
│  ─────────────────                                                     │
│  • Cortisol CRONICAMENTE elevado                                       │
│  • Causado por: estresse crônico, overtraining, déficit                │
│    calórico severo, sono ruim, ansiedade                               │
│  • ESTE sim prejudica ganhos                                           │
│                                                                         │
│  ANALOGIA:                                                             │
│  "Cortisol é como fogo - necessário para cozinhar,                     │
│  mas destrutivo se descontrolado."                                     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

Fonte: Viru & Viru (2004), PMC (2018)
```

### 3.4 Impacto do Cortisol Crônico no Músculo

**Estudo Mendelian Randomization (2022):**
- Cada 1 desvio padrão de aumento em cortisol associado a:
  - Redução de 0.032 SD em força de preensão
  - Redução de 0.032 SD em massa magra total
  - Aumento significativo de massa gorda

**Estudo de inatividade + cortisol (Ferrando et al., 1999):**
- 14 dias de repouso + cortisol elevado
- Aumento de 3x na degradação proteica muscular
- Inatividade AMPLIFICA efeito catabólico do cortisol

## 4. HORMÔNIO DO CRESCIMENTO (GH)

### 4.1 O que é e Como Funciona

GH (Growth Hormone / Somatotropina):

**Funções:**
- Estimula crescimento de tecidos (osso, músculo)
- Promove lipólise (queima de gordura)
- Aumenta síntese proteica
- Estimula produção de IGF-1 no fígado
- Regula metabolismo de glicose

**Secreção:**
- Pulsátil (em ondas)
- ~80% liberado durante sono profundo (SWS)
- Pico nos primeiros 90 minutos de sono
- Também liberado após exercício intenso

### 4.2 GH Pós-Treino: Importa?

```
GH PÓS-TREINO - EVIDÊNCIA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  ESTUDO WEST ET AL. (2010):                                            │
│  ─────────────────────────                                             │
│  • Treino de um braço com protocolo de alto GH (pernas junto)          │
│  • Treino do outro braço com protocolo de baixo GH (só braço)          │
│  • Mediu GH, IGF-1, testosterona                                       │
│  • Mediu hipertrofia após 15 semanas                                   │
│                                                                         │
│  RESULTADO:                                                            │
│  ─────────                                                             │
│  • Protocolo alto GH: 3-4x mais hormônios circulantes                  │
│  • Hipertrofia: SEM DIFERENÇA entre os braços                          │
│  • Força: SEM DIFERENÇA entre os braços                                │
│                                                                         │
│  CONCLUSÃO:                                                            │
│  "Elevações agudas de GH pós-exercício não aumentam                    │
│  hipertrofia ou força."                                                │
│                                                                         │
│  IMPLICAÇÃO PRÁTICA:                                                   │
│  Não precisa fazer treinos "para aumentar GH" - o que                  │
│  importa é volume, intensidade e progressão, não pico hormonal.        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4.3 GH e Sono - A Conexão Real

**Aqui o GH realmente importa:**

| Sono | Liberação de GH | Impacto |
|------|-----------------|---------|
| 7-9h, boa qualidade | Normal (pico no SWS) | Recuperação ótima |
| <6h ou fragmentado | Reduzida 50-70% | Recuperação prejudicada |
| Privação total | Praticamente abolida | Catabolismo |

**Por que GH do sono importa mais que GH pós-treino:**
- GH noturno é liberado em quantidade muito maior
- Coincide com período de jejum (lipólise máxima)
- Janela prolongada de ação (horas)
- Não compete com cortisol elevado

### 4.4 Fatores que Afetam GH

**Aumentam GH:**
- Sono profundo (principal)
- Jejum / baixa glicose sanguínea
- Exercício intenso (agudo)
- Proteína antes de dormir (arginina, glutamina)

**Diminuem GH:**
- Sono ruim/curto
- Obesidade
- Hiperglicemia
- Envelhecimento
- Cortisol cronicamente elevado

## 5. IGF-1 (INSULIN-LIKE GROWTH FACTOR 1)

### 5.1 O que é e Como Funciona

IGF-1 é mediador de muitos efeitos do GH:

**Tipos:**
- **IGF-1 hepático:** Produzido no fígado, circula no sangue
- **IGF-1 muscular (MGF):** Produzido localmente no músculo

**Funções:**
- Estimula síntese proteica via PI3K/Akt/mTOR
- Promove proliferação de células satélite
- Inibe atrofia muscular
- Promove captação de aminoácidos

### 5.2 IGF-1 Local vs Sistêmico

```
IGF-1 LOCAL VS CIRCULANTE:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  IGF-1 CIRCULANTE (do fígado):                                         │
│  ────────────────────────────                                          │
│  • Regulado pelo GH                                                    │
│  • Níveis estáveis ao longo do dia                                     │
│  • NÃO prediz hipertrofia localizada                                   │
│                                                                         │
│  IGF-1 LOCAL (MGF - Mechano Growth Factor):                            │
│  ────────────────────────────────────────────                          │
│  • Produzido NO MÚSCULO em resposta à tensão mecânica                  │
│  • Aumenta após treino do músculo específico                           │
│  • Estimulado pela testosterona local                                  │
│  • ESTE pode ter papel na hipertrofia localizada                       │
│                                                                         │
│  CONCLUSÃO:                                                            │
│  O exercício de força aumenta IGF-1 LOCAL no músculo                   │
│  trabalhado, o que pode contribuir para adaptações.                    │
│  IGF-1 sistêmico é menos relevante.                                    │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 6. INSULINA

### 6.1 O que é e Como Funciona

Insulina é hormônio anabólico e regulador metabólico:

**Funções:**
- Promove captação de glicose pelas células
- Estimula síntese de glicogênio
- Inibe degradação proteica (anti-catabólico)
- Promove captação de aminoácidos
- Estimula lipogênese (em excesso)

### 6.2 Insulina e Síntese Proteica

```
INSULINA E MÚSCULO - A VERDADE:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  PERGUNTA: Preciso de carboidrato pós-treino para                      │
│  maximizar ganhos via insulina?                                        │
│                                                                         │
│  RESPOSTA: NÃO necessariamente.                                        │
│                                                                         │
│  EVIDÊNCIA (Figueiredo & Cameron-Smith, 2013):                         │
│  ─────────────────────────────────────────────                         │
│  • Insulina em níveis fisiológicos NÃO aumenta síntese                 │
│    proteica além do que aminoácidos já proporcionam                    │
│  • Leucina sozinha já estimula insulina suficiente (2-3x)              │
│  • Carboidrato adicional não aumenta MPS além de proteína              │
│  • Insulina REDUZ degradação proteica (efeito anti-catabólico)         │
│  • Mas níveis baixos já são suficientes para esse efeito               │
│                                                                         │
│  QUANDO CARBOIDRATO PÓS-TREINO IMPORTA:                                │
│  ───────────────────────────────────────                               │
│  • Repor glicogênio para próximo treino (<8h)                          │
│  • Atletas com múltiplos treinos/dia                                   │
│  • Em déficit calórico severo (preservar massa)                        │
│                                                                         │
│  CONCLUSÃO PRÁTICA:                                                    │
│  "Para hipertrofia, proteína pós-treino é essencial.                   │
│  Carboidrato pós-treino é útil para energia, mas não                   │
│  amplifica síntese proteica além da proteína sozinha."                 │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 6.3 Sensibilidade à Insulina

| Estado | Sensibilidade | Implicação |
|--------|---------------|------------|
| Atleta treinado | Alta | Menos insulina necessária, partição favorável |
| Sedentário | Moderada | Normal |
| Obeso | Baixa (resistência) | Mais insulina, maior armazenamento de gordura |
| Pós-treino | Muito alta | Janela ótima para nutrientes |

**Como melhorar sensibilidade:**
- Treino de força (principal)
- Cardio/atividade física
- Reduzir gordura corporal
- Sono adequado
- Dieta com fibras, proteína adequada

## 7. ESTROGÊNIO

### 7.1 Papel nas Mulheres

```
ESTROGÊNIO E MÚSCULO FEMININO:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  EFEITOS PROTETORES/ANABÓLICOS:                                        │
│  ──────────────────────────────                                        │
│  • Reduz degradação proteica muscular                                  │
│  • Tem efeitos anti-inflamatórios                                      │
│  • Protege contra dano muscular                                        │
│  • Pode aumentar sensibilidade a estímulos anabólicos                  │
│  • Compensa parcialmente a menor testosterona                          │
│                                                                         │
│  EVIDÊNCIA (HRT em pós-menopausa):                                     │
│  ────────────────────────────────                                      │
│  • Terapia de reposição hormonal reduz perda muscular                  │
│  • Contrabalança upregulation do sistema ubiquitina-proteassoma        │
│  • Melhora expressão de genes miogênicos pós-treino                    │
│                                                                         │
│  IMPLICAÇÃO:                                                           │
│  Estrogênio ajuda a explicar por que mulheres têm ganhos               │
│  RELATIVOS similares aos homens, apesar de muito menos T.              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 7.2 Ciclo Menstrual e Treino

| Fase | Dias | Hormônios | Implicação para Treino |
|------|------|-----------|------------------------|
| **Menstrual** | 1-5 | E↓ P↓ | Energia pode estar baixa |
| **Folicular** | 6-14 | E↑ | Boa energia, boa recuperação |
| **Ovulação** | ~14 | E pico | Pico de energia para algumas |
| **Lútea** | 15-28 | P↑ E↓ | Retenção, possível fadiga |

**Prático:**
- Não há necessidade de periodizar treino pelo ciclo
- Mas algumas mulheres preferem treinos mais intensos na fase folicular
- Monitorar como SE SENTE é mais importante que seguir regras fixas

### 9.2 Checklist para Otimização

```
CHECKLIST - HORMÔNIOS NATURAIS:

□ SONO
  ├─ 7-9 horas por noite
  ├─ Horário consistente (±1h)
  ├─ Quarto escuro e fresco
  └─ Evitar telas 1h antes

□ NUTRIÇÃO
  ├─ Calorias adequadas (não déficit severo crônico)
  ├─ Gordura 20-35% das calorias (colesterol é precursor)
  ├─ Proteína 1.6-2.2 g/kg
  ├─ Zinco adequado (ostras, carne, sementes)
  └─ Vitamina D (sol ou suplemento se deficiente)

□ EXERCÍCIO
  ├─ Treino de força 3-5x/semana
  ├─ Exercícios compostos (agachamento, terra, supino)
  ├─ Progressão de carga
  └─ Evitar overtraining

□ ESTRESSE
  ├─ Técnicas de relaxamento
  ├─ Hobbies e lazer
  ├─ Conexões sociais
  └─ Limitar trabalho excessivo

□ COMPOSIÇÃO CORPORAL
  ├─ Manter BF% saudável (10-20% H, 18-28% M)
  ├─ Evitar obesidade (↓ T)
  └─ Evitar BF% muito baixo cronicamente (↓ T)
```

## 10. IMPLEMENTAÇÃO NO VITRÚVIO

### 10.1 Como Usar Este Conhecimento

```typescript
interface DadosHormonais {
  idade: number
  sexo: 'M' | 'F'
  horasSono: number
  qualidadeSono: number  // 1-10
  estressePercebido: number  // 1-10
  gorduraCorporal: number  // %
  deficitCalorico: boolean
  deficitSevero: boolean  // >500 kcal
  examesDisponiveis?: {
    testosteronaTotal?: number  // ng/dL
    testosteronaLivre?: number  // ng/dL
    cortisol?: number  // μg/dL
    vitaminaD?: number  // ng/mL
  }
}

interface AvaliacaoHormonal {
  statusGeral: 'OTIMIZADO' | 'ADEQUADO' | 'SUBOTIMO' | 'PREOCUPANTE'
  fatoresPositivos: string[]
  fatoresNegativos: string[]
  recomendacoes: string[]
  alertas: string[]
}

function avaliarStatusHormonal(dados: DadosHormonais): AvaliacaoHormonal {
  const resultado: AvaliacaoHormonal = {
    statusGeral: 'ADEQUADO',
    fatoresPositivos: [],
    fatoresNegativos: [],
    recomendacoes: [],
    alertas: []
  }
  
  let score = 100
  
  // Sono
  if (dados.horasSono >= 7 && dados.qualidadeSono >= 7) {
    resultado.fatoresPositivos.push('Sono adequado - suporta GH e T')
  } else if (dados.horasSono < 6) {
    score -= 25
    resultado.fatoresNegativos.push('Sono insuficiente - prejudica GH e T')
    resultado.recomendacoes.push('Priorize dormir 7-9h por noite')
  }
  
  // Estresse
  if (dados.estressePercebido > 7) {
    score -= 15
    resultado.fatoresNegativos.push('Estresse elevado - cortisol cronicamente alto')
    resultado.recomendacoes.push('Implemente técnicas de gerenciamento de estresse')
  }
  
  // Composição corporal
  const bfIdeal = dados.sexo === 'M' ? {min: 10, max: 20} : {min: 18, max: 28}
  if (dados.gorduraCorporal > bfIdeal.max + 10) {
    score -= 20
    resultado.fatoresNegativos.push('Gordura corporal elevada - pode reduzir T')
    resultado.recomendacoes.push('Reduzir gradualmente gordura corporal')
  } else if (dados.gorduraCorporal < bfIdeal.min - 3) {
    score -= 15
    resultado.fatoresNegativos.push('Gordura corporal muito baixa - pode reduzir T')
    resultado.recomendacoes.push('Considere aumentar BF% para faixa saudável')
  }
  
  // Déficit calórico
  if (dados.deficitSevero) {
    score -= 20
    resultado.fatoresNegativos.push('Déficit calórico severo - reduz T e GH')
    resultado.recomendacoes.push('Modere o déficit para -300 a -500 kcal')
  }
  
  // Idade
  if (dados.idade > 40) {
    resultado.recomendacoes.push('Considere check-up hormonal anual')
  }
  if (dados.idade > 50) {
    resultado.recomendacoes.push('Aumente proteína para 2.0-2.4 g/kg')
    resultado.recomendacoes.push('Priorize ainda mais sono e recuperação')
  }
  
  // Exames se disponíveis
  if (dados.examesDisponiveis?.testosteronaTotal) {
    const t = dados.examesDisponiveis.testosteronaTotal
    const minT = dados.sexo === 'M' ? 300 : 15
    const maxT = dados.sexo === 'M' ? 1000 : 70
    
    if (t < minT) {
      score -= 30
      resultado.alertas.push('Testosterona abaixo do normal - consulte endocrinologista')
    } else if (t > minT && t < minT * 1.5) {
      resultado.recomendacoes.push('Testosterona na faixa baixa - otimize fatores de estilo de vida')
    }
  }
  
  if (dados.examesDisponiveis?.vitaminaD) {
    if (dados.examesDisponiveis.vitaminaD < 30) {
      resultado.recomendacoes.push('Vitamina D baixa - considere suplementação')
    }
  }
  
  // Status geral
  if (score >= 85) resultado.statusGeral = 'OTIMIZADO'
  else if (score >= 70) resultado.statusGeral = 'ADEQUADO'
  else if (score >= 50) resultado.statusGeral = 'SUBOTIMO'
  else resultado.statusGeral = 'PREOCUPANTE'
  
  return resultado
}
```

## 11. REGRAS DE OURO - HORMÔNIOS

```
REGRAS DO VITRÚVIO PARA HORMÔNIOS:

1. HORMÔNIOS AGUDOS NÃO PREDIZEM GANHOS
   • Picos de T, GH, IGF-1 pós-treino são normais
   • Mas NÃO aceleram hipertrofia
   • O que importa são níveis basais adequados

2. SONO É O MAIOR OTIMIZADOR HORMONAL
   • 80% do GH durante sono profundo
   • Testosterona pico durante o sono
   • <6h = ambiente catabólico

3. CORTISOL NÃO É O INIMIGO
   • Cortisol agudo é necessário
   • Problema é cortisol CRÔNICO
   • Gerenciar estresse de vida é crucial

4. MULHERES GANHAM IGUAL PROPORCIONALMENTE
   • Menos T, mas ganhos relativos similares
   • Estrogênio tem efeitos protetores
   • Não precisam treinar diferente

5. IDADE IMPORTA, MAS TREINO COMPENSA
   • T e GH caem com idade
   • Treino de força MANTÉM níveis
   • Ajustar proteína e recuperação

6. COMPOSIÇÃO CORPORAL AFETA HORMÔNIOS
   • Obesidade reduz T
   • BF% muito baixo também reduz T
   • Manter faixa saudável

7. DÉFICIT SEVERO PREJUDICA HORMÔNIOS
   • Déficit >500-750 kcal reduz T e GH
   • Moderar déficit preserva hormônios
   • Diet breaks podem ajudar

8. RECEPTOR É MAIS IMPORTANTE QUE HORMÔNIO
   • Conteúdo de AR no músculo prediz ganhos
   • Treino aumenta AR (upregulation)
   • Por isso treino funciona mesmo com T "normal"
```
