## 1. VISÃO GERAL

### 1.1 O que é esta SPEC?

Esta SPEC contém o conhecimento científico sobre **frequência de treino** que o VITRÚVIO deve usar como base para:

- Determinar quantas vezes treinar cada grupo muscular por semana
- Distribuir volume de treino entre sessões
- Escolher o tipo de divisão de treino (split)
- Otimizar a síntese proteica muscular através da frequência

### 1.2 Definições

```
TIPOS DE FREQUÊNCIA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  FREQUÊNCIA DE SESSÕES                                                 │
│  ─────────────────────                                                 │
│  Número total de treinos por semana                                    │
│  Exemplo: "Treino 4 vezes por semana"                                  │
│                                                                         │

│  ─────────────────────────────────                                     │
│  Quantas vezes cada músculo é treinado por semana                      │
```

## 2. A CIÊNCIA DA FREQUÊNCIA

### 2.1 Síntese Proteica Muscular (MPS)

```
DURAÇÃO DA SÍNTESE PROTEICA APÓS TREINO:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  TIMELINE DA MPS:                                                      │
│  ─────────────────                                                     │
│                                                                         │
│  0-4 horas:     MPS aumenta ~50% acima do baseline                     │
│  24 horas:      MPS no PICO (~100-150% acima do baseline)              │
│  36 horas:      MPS ainda elevada mas diminuindo                       │
│  48 horas:      MPS retornando ao baseline (em treinados)              │
│  72 horas:      MPS próxima ou no baseline                             │
│                                                                         │
│  EM NÃO TREINADOS:                                                     │
│  • MPS pode ficar elevada por 72-96 horas                              │
│  • Porém, muito dessa MPS é para REPARAR dano muscular                 │
│  • Não necessariamente para crescimento                                │
│                                                                         │
│  EM TREINADOS:                                                         │
│  • MPS retorna ao baseline mais rápido (24-48h)                        │
│  • Menos dano muscular = mais MPS para crescimento                     │
│  • Portanto, frequência MAIOR pode ser benéfica                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 3. META-ANÁLISES E EVIDÊNCIAS

### 3.1 Schoenfeld et al. (2016) - Primeira Meta-Análise

**Estudo:** "Effects of Resistance Training Frequency on Measures of Muscle Hypertrophy: A Systematic Review and Meta-Analysis"

**Metodologia:**
- 10 estudos que compararam diferentes frequências
- Análise de hipertrofia (medidas diretas e indiretas)
- Volume NÃO equalizado entre grupos

**Resultados:**

| Frequência | Effect Size | p-value |
|------------|-------------|---------|
| Maior (2-3x/sem) | 0.49 ± 0.08 | - |
| Menor (1x/sem) | 0.30 ± 0.07 | - |
| Diferença | Significativa | p = 0.002 |

**Conclusão:** "Frequências de pelo menos 2x/semana promovem hipertrofia superior a 1x/semana."

### 3.2 Schoenfeld et al. (2019) - Segunda Meta-Análise (Volume Equalizado)

**Estudo:** "How many times per week should a muscle be trained to maximize muscle hypertrophy?"

**Metodologia:**
- 25 estudos incluídos
- **Volume EQUALIZADO** entre grupos (diferença era apenas frequência)
- Análises por subgrupos (treinados, destreinados, medidas diretas/indiretas)

**Resultados:**

| Análise | Diferença | Significância |
|---------|-----------|---------------|
| Geral (volume equalizado) | Sem diferença significativa | p > 0.05 |
| Medidas diretas de hipertrofia | Sem diferença significativa | p > 0.05 |
| Indivíduos treinados | Sem diferença significativa | p > 0.05 |

**Conclusão:** "Quando o volume é equalizado, não há diferença significativa entre frequências. Indivíduos podem escolher frequência baseada em preferência pessoal."

### 3.3 Reconciliando os Dois Estudos

```
POR QUE OS RESULTADOS PARECEM CONTRADITÓRIOS?

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  META-ANÁLISE 2016 (volume NÃO equalizado):                            │
│  ─────────────────────────────────────────                             │
│  • Grupos de maior frequência também tinham MAIS VOLUME                │
│  • Exemplo: 3x/semana = 9 séries totais vs 1x/semana = 3 séries        │
│  • Não é surpresa que mais volume = mais hipertrofia                   │
│  • A frequência foi confundida com volume                              │
│                                                                         │
│  META-ANÁLISE 2019 (volume EQUALIZADO):                                │
│  ──────────────────────────────────────                                │
│  • Mesmo volume total, apenas frequência diferente                     │
│  • Exemplo: 9 séries em 1 dia vs 3 séries em 3 dias                    │
│  • Resultado: sem diferença significativa                              │
│                                                                         │
│  CONCLUSÃO PRÁTICA:                                                    │
│  ─────────────────────                                                 │
│  1. VOLUME TOTAL é o que mais importa                                  │
│  2. Frequência é uma forma de DISTRIBUIR o volume                      │
│  3. Maior frequência PERMITE maior volume (mais fácil de acumular)     │
│  4. Mas se volume é igual, frequência não faz diferença significativa  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.4 Análise do Stronger by Science (2020)

Greg Nuckols reanalisou os dados com metodologia mais rigorosa:

```
RESULTADOS DA REANÁLISE:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  13 estudos, 305 sujeitos, 40 comparações                              │
│                                                                         │
│  ACHADOS:                                                              │
│  • Maior frequência = mais hipertrofia (p < 0.0001)                    │
│  • Porém, efeito é TRIVIAL (d = 0.113)                                 │
│  • Hipertrofia semanal: ~0.02% maior com frequência alta               │
│                                                                         │
│  EM TERMOS PRÁTICOS:                                                   │
│  • Diferença de ~0.5-1% ao longo de 1 ano                              │
│  • Quase irrelevante para maioria dos praticantes                      │
│  • Outros fatores (aderência, preferência) mais importantes            │
│                                                                         │
│  NUANCE IMPORTANTE:                                                    │
│  • Volume ALTO pode se beneficiar mais de frequência alta              │
│  • Razão: difícil fazer 20+ séries de qualidade em 1 sessão            │
│  • Dividir em 2-3 sessões permite melhor qualidade por série           │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 4. FREQUÊNCIA ÓTIMA POR GRUPO MUSCULAR

### 4.1 Recomendação Geral

```
FREQUÊNCIA RECOMENDADA POR GRUPO MUSCULAR:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  MÍNIMO: 1x/semana                                                     │
│  • Funciona, mas provavelmente subótimo                                │
│  • Pode ser suficiente para manutenção                                 │
│                                                                         │
│  IDEAL: 2x/semana                                                      │
│  • Melhor balanço entre estímulo e recuperação                         │
│  • Permite boa distribuição de volume                                  │
│  • Recomendado para maioria dos praticantes                            │
│                                                                         │
│  POSSÍVEL: 3x/semana                                                   │
│  • Pode funcionar bem com volume moderado por sessão                   │
│  • Útil para grupos musculares prioritários                            │
│  • Requer boa capacidade de recuperação                                │
│                                                                         │
│  MÁXIMO: 4-6x/semana (mesmo músculo)                                   │
│  • Apenas para atletas avançados                                       │
│  • Volume muito baixo por sessão                                       │
│  • Usado em programas específicos (daily undulating periodization)     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 6. VOLUME POR SESSÃO vs FREQUÊNCIA

### 6.1 O Trade-off

```
RELAÇÃO VOLUME-FREQUÊNCIA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  Regra fundamental:                                                    │
│  VOLUME SEMANAL = SÉRIES POR SESSÃO × FREQUÊNCIA SEMANAL               │
│                                                                         │
│  EXEMPLOS (16 séries de peito por semana):                             │
│  ─────────────────────────────────────────────                         │
│                                                                         │
│  1x/semana: 16 séries em 1 sessão                                      │
│  → Fadiga extrema nas últimas séries                                   │
│  → Qualidade das séries 12-16 muito baixa                              │
│                                                                         │
│  2x/semana: 8 séries por sessão                                        │
│  → Boa qualidade em todas as séries                                    │
│  → Recuperação adequada entre sessões                                  │
│  → RECOMENDADO                                                         │
│                                                                         │
│  3x/semana: 5-6 séries por sessão                                      │
│  → Excelente qualidade por série                                       │
│  → Requer mais dias no ginásio                                         │
│  → Bom para prioridades ou atletas avançados                           │
│                                                                         │
│  4x/semana: 4 séries por sessão                                        │
│  → Muito específico, pode ser redundante                               │
│  → Poucos ganhos adicionais sobre 3x                                   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 6.2 Limite de Volume por Sessão

```
VOLUME MÁXIMO EFETIVO POR SESSÃO:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  REGRA GERAL:                                                          │
│  • Máximo ~6-10 séries por grupo muscular por sessão                   │
│  • Após 8 séries, cada série adicional tem retornos muito diminuídos   │
│  • A partir de 10 séries, pode até ser contraproducente                │
│                                                                         │
│  IMPLICAÇÃO:                                                           │
│  Se você quer fazer mais de 10 séries semanais para um grupo,          │
│  é MELHOR dividir em 2+ sessões.                                       │
│                                                                         │
│  EXEMPLO:                                                              │
│  ──────────                                                            │
│  Meta: 16 séries de quadríceps por semana                              │
│                                                                         │
│  Opção A (1x/semana):                                                  │
│  • 16 séries em 1 dia                                                  │
│  • Séries 1-6: alta qualidade                                          │
│  • Séries 7-10: qualidade moderada                                     │
│  • Séries 11-16: fadiga extrema, pouco estímulo efetivo                │
│                                                                         │
│  Opção B (2x/semana):                                                  │
│  • 8 séries em 2 dias diferentes                                       │
│  • Todas as 16 séries com boa qualidade                                │
│  • Melhor resultado com mesmo volume total                             │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 8. COMO O VITRÚVIO DEVE PRESCREVER

### 8.1 Algoritmo de Prescrição

```typescript
interface ParametrosFrequencia {
  experiencia: 'INICIANTE' | 'INTERMEDIARIO' | 'AVANCADO' | 'ELITE'
  diasDisponiveis: number  // 2-7
  tempoSessao: number      // minutos disponíveis
  objetivo: 'HIPERTROFIA' | 'FORCA' | 'MANUTENCAO'
  gruposPrioritarios: string[]
  capacidadeRecuperacao: 'BAIXA' | 'MEDIA' | 'ALTA'
}

interface PlanoFrequencia {
  split: string
  diasPorSemana: number
  frequenciaPorGrupo: Record<string, number>
  volumePorSessao: Record<string, number>
  justificativa: string[]
}

function determinarFrequencia(params: ParametrosFrequencia): PlanoFrequencia {
  let split: string
  let frequenciaBase = 2  // default 2x/semana por grupo
  
  // Determinar split baseado em dias disponíveis
  if (params.diasDisponiveis <= 3) {
    split = 'FULL_BODY'
    frequenciaBase = params.diasDisponiveis
  } else if (params.diasDisponiveis === 4) {
    split = 'UPPER_LOWER'
    frequenciaBase = 2
  } else if (params.diasDisponiveis === 5) {
    split = 'UPPER_LOWER_PUSH_PULL_LEGS'
    frequenciaBase = 2  // com ajustes
  } else {
    split = 'PUSH_PULL_LEGS'
    frequenciaBase = 2
  }
  
  // Ajustar por experiência
  if (params.experiencia === 'INICIANTE') {
    split = 'FULL_BODY'
    frequenciaBase = Math.min(params.diasDisponiveis, 3)
  }
  
  // Ajustar por capacidade de recuperação
  if (params.capacidadeRecuperacao === 'BAIXA') {
    frequenciaBase = Math.min(frequenciaBase, 2)
  } else if (params.capacidadeRecuperacao === 'ALTA') {
    // Pode aumentar para grupos prioritários
  }
  
  // Calcular frequência por grupo
  const frequenciaPorGrupo: Record<string, number> = {
    'PEITO': frequenciaBase,
    'COSTAS': frequenciaBase,
    'OMBROS': frequenciaBase,
    'BICEPS': frequenciaBase,
    'TRICEPS': frequenciaBase,
    'QUADRICEPS': frequenciaBase,
    'ISQUIOTIBIAIS': frequenciaBase,
    'GLUTEOS': frequenciaBase,
    'PANTURRILHA': Math.min(frequenciaBase + 1, 4),  // tolera mais
    'ABDOMINAIS': Math.min(frequenciaBase + 1, 4)
  }
  
  // Aumentar frequência de grupos prioritários
  for (const grupo of params.gruposPrioritarios) {
    if (frequenciaPorGrupo[grupo] && params.capacidadeRecuperacao !== 'BAIXA') {
      frequenciaPorGrupo[grupo] = Math.min(frequenciaPorGrupo[grupo] + 1, 4)
    }
  }
  
  // Calcular volume por sessão (assumindo metas de volume semanal)
  const volumeSemanalPorGrupo: Record<string, number> = {
    'PEITO': 12,
    'COSTAS': 14,
    'OMBROS': 10,
    'BICEPS': 10,
    'TRICEPS': 10,
    'QUADRICEPS': 14,
    'ISQUIOTIBIAIS': 10,
    'GLUTEOS': 10,
    'PANTURRILHA': 12,
    'ABDOMINAIS': 8
  }
  
  const volumePorSessao: Record<string, number> = {}
  for (const grupo in frequenciaPorGrupo) {
    volumePorSessao[grupo] = Math.round(
      volumeSemanalPorGrupo[grupo] / frequenciaPorGrupo[grupo]
    )
  }
  
  // Preparar justificativas
  const justificativa: string[] = [
    `Split ${split} escolhido para ${params.diasDisponiveis} dias disponíveis`,
    `Frequência de ${frequenciaBase}x/semana para maioria dos grupos`
  ]
  
  if (params.gruposPrioritarios.length > 0) {
    justificativa.push(
      `Frequência aumentada para grupos prioritários: ${params.gruposPrioritarios.join(', ')}`
    )
  }
  
  return {
    split,
    diasPorSemana: params.diasDisponiveis,
    frequenciaPorGrupo,
    volumePorSessao,
    justificativa
  }
}
```

### 9.2 Regras de Ouro

```
REGRAS DO VITRÚVIO PARA FREQUÊNCIA:

1. VOLUME TOTAL É MAIS IMPORTANTE QUE FREQUÊNCIA
   • Se volume é igual, frequência faz pouca diferença
   • Frequência é uma forma de DISTRIBUIR volume

2. MÍNIMO 2X/SEMANA POR GRUPO PARA OTIMIZAR
   • 1x/semana funciona, mas 2x é melhor para maioria
   • Exceto se volume for muito baixo

3. MÁXIMO ~8-10 SÉRIES POR GRUPO POR SESSÃO
   • Acima disso, dividir em mais sessões
   • Qualidade > quantidade em uma única sessão

4. 48 HORAS MÍNIMO ENTRE SESSÕES DO MESMO GRUPO
   • Permite recuperação adequada
   • Exceto com volume muito baixo por sessão

5. FREQUÊNCIA ALTA PARA GRUPOS "TEIMOSOS"
   • Panturrilha, deltoide posterior, abs
   • Toleram e podem precisar de mais estímulo

6. ESCOLHER SPLIT BASEADO EM DISPONIBILIDADE
   • O melhor split é o que você vai FAZER consistentemente
   • Não adianta prescrever 6x/semana se pessoa só tem 4 dias

7. INICIANTES: FULL BODY OU ALTA FREQUÊNCIA
   • Aprender técnica com repetição frequente
   • Volume baixo por sessão, frequência alta
```
