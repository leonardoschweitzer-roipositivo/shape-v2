## 1. VISÃO GERAL

### 1.1 O que é esta SPEC?

Esta SPEC contém o conhecimento científico sobre **volume de treino** que o VITRÚVIO deve usar como base para:

- Prescrever número de séries por grupo muscular
- Ajustar volume conforme nível do atleta
- Identificar volume mínimo, ótimo e máximo
- Periodizar volume ao longo do tempo

### 1.2 Definição de Volume

> **Volume de Treino** = Número total de séries desafiadoras (hard sets) realizadas por grupo muscular em um período de tempo (geralmente por semana).

```
FORMAS DE MEDIR VOLUME:

1. Séries por grupo muscular por semana (mais usado)
   Exemplo: 15 séries de peitoral/semana

2. Séries × Repetições × Carga (tonnage)
   Exemplo: 10 séries × 10 reps × 80kg = 8.000 kg

3. Séries × Repetições (volume-load)
   Exemplo: 10 séries × 10 reps = 100 repetições
```

## 2. CONCEITOS FUNDAMENTAIS: MEV, MAV, MRV

### 2.1 Definições (Renaissance Periodization / Mike Israetel)

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  MV (Maintenance Volume)                                               │
│  ─────────────────────────                                             │
│  Volume mínimo para MANTER o tamanho muscular atual.                   │
│  Geralmente: 4-6 séries/semana por grupo muscular                      │
│                                                                         │
│  MEV (Minimum Effective Volume)                                        │
│  ─────────────────────────────────                                     │
│  Volume mínimo para PRODUZIR crescimento muscular.                     │
│  Geralmente: 6-8 séries/semana por grupo muscular                      │
│                                                                         │
│  MAV (Maximum Adaptive Volume)                                         │
│  ─────────────────────────────────                                     │
│  Volume que produz o MÁXIMO de crescimento por unidade de esforço.     │
│  Zona ótima de custo-benefício.                                        │
│  Geralmente: 10-20 séries/semana por grupo muscular                    │
│                                                                         │
│  MRV (Maximum Recoverable Volume)                                      │
│  ─────────────────────────────────                                     │
│  Volume MÁXIMO do qual você ainda consegue se recuperar.               │
│  Acima disso: overreaching/overtraining.                               │
│  Geralmente: 20-30+ séries/semana (varia muito individualmente)        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 2.3 Curva de Dose-Response

Baseado na meta-análise de Schoenfeld et al. (2017):

```
GANHOS DE HIPERTROFIA POR VOLUME SEMANAL:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  < 5 séries/semana:     Ganhos mínimos ou subótimos                    │
│                                                                         │
│  5-9 séries/semana:     Ganhos moderados                               │
│                         (~0.37% adicional por série)                   │
│                                                                         │
│  10-20 séries/semana:   Ganhos ótimos para maioria                     │
│                         (ponto ideal custo-benefício)                  │
│                                                                         │
│  20+ séries/semana:     Retornos diminuídos                            │
│                         (pode beneficiar atletas avançados)            │
│                                                                         │
│  30+ séries/semana:     Risco de overreaching                          │
│                         (apenas temporariamente, com deload)           │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

Cada série adicional ≈ +0.37% de ganho em hipertrofia
(Schoenfeld et al., 2017)
```

## 3. META-ANÁLISES E EVIDÊNCIAS

### 3.1 Schoenfeld et al. (2017) - Dose-Response

**Estudo:** "Dose-response relationship between weekly resistance training volume and increases in muscle mass"

**Metodologia:**
- Meta-análise de 15 estudos
- 34 grupos de tratamento
- Análise de volume semanal como variável contínua

**Resultados principais:**

| Volume Semanal | Effect Size | Ganho % |
|----------------|-------------|---------|
| < 5 séries | 0.20 | Baixo |
| 5-9 séries | 0.32 | Moderado |
| 10+ séries | 0.44 | Alto |

**Conclusão:** Relação dose-response gradual - mais volume = mais hipertrofia (até certo ponto).

### 3.2 Schoenfeld et al. (2019) - Frequência

**Estudo:** "How many times per week should a muscle be trained to maximize muscle hypertrophy?"

**Resultados:**
- Quando volume é equalizado, frequência não importa significativamente
- 2x/semana por grupo muscular é suficiente
- 3x/semana não mostrou vantagem sobre 2x (volume igual)

**Implicação prática:** Distribua o volume em 2-3 sessões por grupo muscular.

### 3.3 Baz-Valle et al. (2022) - Volume Moderado vs Alto

**Estudo:** "A Systematic Review of The Effects of Different Resistance Training Volumes on Muscle Hypertrophy"

**Metodologia:**
- Comparação: 12-20 séries (moderado) vs 20+ séries (alto)
- 7 estudos incluídos

**Resultados por grupo muscular:**

| Grupo | Moderado vs Alto | Conclusão |
|-------|------------------|-----------|
| Quadríceps | p = 0.19 | Sem diferença significativa |
| Bíceps | p = 0.59 | Sem diferença significativa |
| Tríceps | p = 0.01 | Alto volume MELHOR |

**Conclusão:** 12-20 séries/semana é ótimo para maioria. Tríceps pode precisar de mais volume (músculo sinergista).

### 3.4 Krieger (2020) - Volume por Sessão

**Estudo atualizado:** Análise de volume por sessão de treino

**Resultados:**
- Hipertrofia aumenta até ~6-8 séries por grupo muscular por SESSÃO
- Acima de 8 séries/sessão: retornos diminuídos significativos
- "First set is most effective, subsequent sets have diminishing returns"

**Recomendação:** Máximo de 6-8 séries por grupo muscular por treino.

### 3.5 Pelland et al. (2024) - Modelo Atualizado

**Estudo:** "The Resistance Training Dose Response: Meta-Regressions"

**Metodologia:**
- 67 estudos, 2058 participantes
- Distinção entre séries diretas e indiretas
- Modelo de retornos diminuídos (não linear)

**Resultados:**
- Probabilidade de 100% que mais volume = mais hipertrofia
- Relação logarítmica (retornos diminuídos)
- Força mostra retornos mais diminuídos que hipertrofia

│                                                                         │

│  • Multiplicador: 1.0x (valores ótimos da tabela)                      │

│  • Pode periodizar volume                                              │
│                                                                         │
│  AVANÇADO (3-7 anos):                                                  │
│  • Multiplicador: 1.0-1.2x                                             │

```

### 4.3 Considerações por Grupo Muscular

**Grupos que toleram MAIS volume:**
- Quadríceps (grande massa, recuperação rápida)
- Costas/dorsais (grande área)
- Ombros (deltoide anterior/lateral)
- Panturrilha (fibras tipo I, alta resistência)

**Grupos que toleram MENOS volume:**
- Isquiotibiais (propensos a lesão)
- Bíceps (pequeno, fácil overtraining)
- Lombar/eretores (estresse no sistema nervoso)
- Trapézio (recebe estímulo de muitos exercícios)

**Grupos que PRECISAM de mais volume:**
- Panturrilha (fibras tipo I resistentes)
- Tríceps (sinergista em muitos exercícios, precisa de isolamento extra)
- Deltoide posterior (frequentemente subdesenvolvido)

## 6. SÉRIES DIRETAS vs INDIRETAS

### 6.1 Definição

```
SÉRIES DIRETAS:
• Exercícios que trabalham o músculo como AGONISTA principal
• Exemplo: Supino para peitoral, Rosca para bíceps

SÉRIES INDIRETAS:
• Exercícios que trabalham o músculo como SINERGISTA
• Exemplo: Supino para tríceps, Remada para bíceps

CONTAGEM (modelo de Pelland et al., 2024):
• Séries diretas: contam 100%
• Séries indiretas: contam ~50%
• Ou ignorar séries indiretas e prescrever isolamento extra
```

### 6.2 Exemplo Prático

```
TREINO DE PEITO E TRÍCEPS:

Exercício              Peito        Tríceps
─────────────────────────────────────────────
Supino Reto 4×8        4 diretas    4 indiretas (~2)
Supino Inclinado 3×10  3 diretas    3 indiretas (~1.5)
Crucifixo 3×12         3 diretas    0
Tríceps Pulley 4×12    0            4 diretas
Tríceps Francês 3×10   0            3 diretas
─────────────────────────────────────────────
TOTAL SESSÃO:          10 diretas   7 diretas + 3.5 indiretas
                                    = ~10.5 séries equivalentes

Se treinar peito/tríceps 2x/semana:
• Peito: 20 séries/semana (ótimo)
• Tríceps: ~21 séries/semana (alto - pode reduzir isolamento)
```

### 6.3 Implicações Práticas

```
MÚSCULOS QUE RECEBEM MUITO VOLUME INDIRETO:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  TRÍCEPS:                                                              │
│  • Todo exercício de empurrar trabalha tríceps                         │
│  • Supino, desenvolvimento, paralelas                                  │
│  • Pode precisar de MENOS isolamento se volume de empurrar é alto      │
│                                                                         │
│  BÍCEPS:                                                               │
│  • Todo exercício de puxar trabalha bíceps                             │

│  • Pode precisar de MENOS isolamento se volume de puxar é alto         │
│                                                                         │
│  DELTOIDE ANTERIOR:                                                    │
│  • Todo supino trabalha deltoide anterior                              │
│  • Raramente precisa de isolamento direto                              │
│                                                                         │
│  DELTOIDE POSTERIOR:                                                   │
│  • Trabalha em remadas, mas não suficientemente                        │

## 8. COMO O VITRÚVIO DEVE PRESCREVER VOLUME

### 8.1 Algoritmo de Prescrição

```typescript
interface ParametrosVolume {
  experiencia: 'INICIANTE' | 'INTERMEDIARIO' | 'AVANCADO' | 'ELITE'
  grupoMuscular: GrupoMuscular
  objetivo: 'MANUTENCAO' | 'CRESCIMENTO' | 'ESPECIALIZACAO'
  frequenciaSemanal: number  // quantas vezes treina o grupo por semana
  fatoresIndividuais: {
    recuperacao: 'BAIXA' | 'MEDIA' | 'ALTA'
    sono: number  // horas por noite
    estresse: 'BAIXO' | 'MEDIO' | 'ALTO'
    idade: number
    usaHormonios: boolean
  }
}

interface VolumeRecomendado {
  seriasSemanais: number
  seriesPorSessao: number
  rangeMin: number
  rangeMax: number
  notas: string[]
}

// Volume base por grupo muscular (intermediário, objetivo crescimento)
const VOLUME_BASE: Record<GrupoMuscular, { mev: number, mav: number, mrv: number }> = {
  PEITO:            { mev: 8,  mav: 15, mrv: 22 },
  COSTAS_LARGURA:   { mev: 8,  mav: 15, mrv: 22 },
  COSTAS_ESPESSURA: { mev: 6,  mav: 13, mrv: 20 },
  OMBRO_LATERAL:    { mev: 8,  mav: 15, mrv: 23 },
  OMBRO_POSTERIOR:  { mev: 4,  mav: 11, mrv: 18 },
  BICEPS:           { mev: 5,  mav: 13, mrv: 20 },
  TRICEPS:          { mev: 5,  mav: 13, mrv: 21 },
  QUADRICEPS:       { mev: 8,  mav: 15, mrv: 25 },
  ISQUIOTIBIAIS:    { mev: 5,  mav: 12, mrv: 18 },
  GLUTEOS:          { mev: 4,  mav: 11, mrv: 19 },
  PANTURRILHA:      { mev: 8,  mav: 14, mrv: 23 },
  ABDOMINAIS:       { mev: 4,  mav: 11, mrv: 18 },
  TRAPEZIO:         { mev: 4,  mav: 10, mrv: 16 },
  ANTEBRACO:        { mev: 4,  mav: 9,  mrv: 16 }
}

// Multiplicadores por experiência
const MULT_EXPERIENCIA = {
  INICIANTE: 0.7,
  INTERMEDIARIO: 1.0,
  AVANCADO: 1.15,
  ELITE: 1.3
}

// Multiplicadores por objetivo
const MULT_OBJETIVO = {
  MANUTENCAO: 0.5,      // apenas MEV
  CRESCIMENTO: 1.0,     // MAV
  ESPECIALIZACAO: 1.2   // acima do MAV para grupo prioritário
}

function calcularVolume(params: ParametrosVolume): VolumeRecomendado {
  const base = VOLUME_BASE[params.grupoMuscular]
  const multExp = MULT_EXPERIENCIA[params.experiencia]
  const multObj = MULT_OBJETIVO[params.objetivo]
  
  // Calcular volume ideal base
  let volumeIdeal = base.mav * multExp * multObj
  
  // Ajustes por fatores individuais
  if (params.fatoresIndividuais.recuperacao === 'BAIXA') volumeIdeal *= 0.85
  if (params.fatoresIndividuais.recuperacao === 'ALTA') volumeIdeal *= 1.10
  
  if (params.fatoresIndividuais.sono < 6) volumeIdeal *= 0.85
  if (params.fatoresIndividuais.sono >= 8) volumeIdeal *= 1.05
  
  if (params.fatoresIndividuais.estresse === 'ALTO') volumeIdeal *= 0.90
  
  if (params.fatoresIndividuais.idade > 40) volumeIdeal *= 0.95
  if (params.fatoresIndividuais.idade > 50) volumeIdeal *= 0.90
  
  if (params.fatoresIndividuais.usaHormonios) volumeIdeal *= 1.20
  
  // Arredondar e limitar
  volumeIdeal = Math.round(volumeIdeal)
  volumeIdeal = Math.max(base.mev, Math.min(volumeIdeal, base.mrv))
  
  // Calcular séries por sessão
  const seriesPorSessao = Math.min(
    Math.round(volumeIdeal / params.frequenciaSemanal),
    10  // máximo 10 séries por sessão
  )
  
  // Ajustar volume semanal baseado no limite por sessão
  const volumeAjustado = seriesPorSessao * params.frequenciaSemanal
  
  // Preparar notas
  const notas: string[] = []
  if (volumeAjustado < volumeIdeal) {
    notas.push(`Aumente frequência para ${Math.ceil(volumeIdeal / 8)}x/semana para atingir volume ideal`)
  }
  if (params.fatoresIndividuais.sono < 7) {
    notas.push('Volume reduzido devido a sono insuficiente')
  }
  if (params.objetivo === 'ESPECIALIZACAO') {
    notas.push('Volume acima do normal - monitore sinais de overreaching')
  }
  
  return {
    seriasSemanais: volumeAjustado,
    seriesPorSessao,
    rangeMin: Math.round(base.mev * multExp),
    rangeMax: Math.round(base.mrv * multExp),
    notas
  }
}
```
