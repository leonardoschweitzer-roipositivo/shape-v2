## 1. VISÃO GERAL

### 1.1 O que é esta SPEC?

Esta SPEC contém o conhecimento científico sobre **gorduras dietéticas e seu papel na hipertrofia, hormônios e saúde** que o VITRÚVIO deve usar para:

- Orientar quantidade mínima e ideal de gorduras
- Explicar impacto das gorduras na testosterona
- Recomendar tipos de gordura por objetivo
- Orientar sobre ômega-3 e recuperação
- Desmistificar mitos sobre gorduras saturadas

### 1.2 Funções das Gorduras

```
FUNÇÕES DAS GORDURAS DIETÉTICAS:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  1. PRODUÇÃO HORMONAL:                                                 │
│     • Colesterol é precursor de testosterona                           │
│     • Essencial para hormônios esteroides                              │
│     • Low-fat → pode reduzir testosterona                              │
│                                                                         │
│  2. ABSORÇÃO DE VITAMINAS:                                             │
│     • Vitaminas lipossolúveis (A, D, E, K)                             │
│     • Sem gordura, absorção é prejudicada                              │
│                                                                         │
│  3. ESTRUTURA CELULAR:                                                 │
│     • Membranas celulares são feitas de lipídios                       │
│     • Afeta sinalização celular e função                               │
│                                                                         │
│  4. ENERGIA:                                                           │
│     • 1g de gordura = 9 kcal (mais denso que carbs/proteína)           │
│     • Reserva energética do corpo                                      │
│                                                                         │
│  5. SACIEDADE:                                                         │
│     • Gordura retarda esvaziamento gástrico                            │
│     • Ajuda a controlar apetite                                        │
│                                                                         │
│  6. INFLAMAÇÃO:                                                        │
│     • Ômega-3: anti-inflamatório                                       │
│     • Ômega-6 em excesso: pró-inflamatório                             │
│     • Balanço é importante para recuperação                            │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.3 Gorduras são Essenciais

Diferente de carboidratos, algumas gorduras são **essenciais** - o corpo não produz:

| Ácido Graxo | Essencial? | Função Principal |
|-------------|------------|------------------|
| Ômega-3 (EPA, DHA) | **SIM** | Anti-inflamatório, cérebro, coração |
| Ômega-6 (LA) | **SIM** | Estrutura celular, mas em excesso inflama |
| Ômega-9 | NÃO | Corpo produz, mas benéfico |
| Saturadas | NÃO | Corpo produz, mas necessário na dieta |

## 3. GORDURAS E TESTOSTERONA

### 3.1 A Conexão Gordura-Hormônios

```
GORDURA E TESTOSTERONA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  META-ANÁLISE WHITTAKER & WU (2021):                                   │
│  ────────────────────────────────────                                  │
│  • 6 estudos de intervenção, 206 participantes                         │
│  • Comparou dietas low-fat vs high-fat                                 │
│                                                                         │
│  RESULTADOS:                                                           │
│  ───────────                                                           │
│  • Testosterona total: REDUÇÃO significativa em low-fat                │
│  • Testosterona livre: REDUÇÃO significativa em low-fat                │
│  • DHT: REDUÇÃO significativa em low-fat                               │
│  • Efeito mais forte em homens europeus/norte-americanos               │
│                                                                         │
│  MAGNITUDE:                                                            │
│  ──────────                                                            │
│  • Diferença de ~10-15% na testosterona                                │
│  • Low-fat = geralmente <20-25% das calorias de gordura                │
│                                                                         │
│  POR QUE ISSO ACONTECE:                                                │
│  ──────────────────────                                                │
│  • Colesterol é precursor da testosterona                              │
│  • Gordura saturada relacionada à produção hormonal                    │
│  • Déficit calórico também reduz T (confundidor)                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 Qual Gordura Afeta Mais a Testosterona?

| Tipo de Gordura | Efeito na Testosterona | Evidência |
|-----------------|------------------------|-----------|
| **Saturada** | Pode AUMENTAR T | Moderada |
| **Monoinsaturada** | Neutro a positivo | Moderada |
| **Poli-insaturada (PUFA)** | Neutro a REDUZIR T | Fraca-Moderada |
| **Trans** | Pode REDUZIR T | Limitada |

**Interpretação:**
- Gordura saturada tem correlação positiva com testosterona
- Mas excesso de PUFA (especialmente ômega-6) pode reduzir
- Balanço é importante - não eliminar saturada completamente

### 3.3 Recomendação para Otimizar Testosterona


## 4. ÔMEGA-3

### 4.1 O que são EPA e DHA

**Tipos de Ômega-3:**

| Tipo | Nome | Fonte | Conversão |
|------|------|-------|-----------|
| **ALA** | Ácido alfa-linolênico | Linhaça, chia, nozes | Precursor (conversão baixa) |
| **EPA** | Ácido eicosapentaenoico | Peixes gordos | Ativo (anti-inflamatório) |
| **DHA** | Ácido docosahexaenoico | Peixes gordos | Ativo (cérebro, membranas) |

**Importante:** Conversão de ALA → EPA/DHA é muito baixa:
- Homens: <8% para EPA, <4% para DHA
- Mulheres: ~21% para EPA, ~9% para DHA (estrogênio ajuda)

**Conclusão:** Preferir fontes diretas de EPA/DHA (peixes) ou suplementar.

### 4.2 Ômega-3 e Hipertrofia

```
ÔMEGA-3 E MÚSCULO - EVIDÊNCIA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  MECANISMOS POTENCIAIS:                                                │
│  ──────────────────────                                                │
│  • Aumenta sensibilidade da síntese proteica a aminoácidos             │
│  • Ativa via mTOR/p70S6K                                               │
│  • Reduz inflamação crônica (pode atenuar anabolismo)                  │
│  • Melhora fluidez das membranas celulares                             │
│  • Reduz degradação proteica                                           │
│                                                                         │
│  ESTUDO SMITH ET AL. (2011):                                           │
│  ──────────────────────────                                            │
│  • 8 semanas de suplementação com óleo de peixe (4g/dia)               │
│  • AUMENTOU resposta de síntese proteica a aminoácidos                 │
│  • Efeito tanto em jovens quanto em idosos                             │
│                                                                         │
│  META-ANÁLISE RECENTE (2024-2025):                                     │
│  ─────────────────────────────────                                     │
│  • Ômega-3 pode aumentar MPS em algumas condições                      │
│  • Mas NÃO há evidência forte de maior hipertrofia                     │
│  • Benefício mais claro: recuperação e anti-inflamação                 │
│                                                                         │
│  CONCLUSÃO:                                                            │
│  ──────────                                                            │
│  Ômega-3 provavelmente NÃO é "anabólico" direto para quem             │
│  já consome proteína adequada. Mas ajuda na recuperação,               │
│  especialmente em idosos e em situações de dano muscular.              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4.3 Ômega-3 e Recuperação

| Efeito | Evidência | Implicação Prática |
|--------|-----------|-------------------|
| **Reduz inflamação** | Forte | Pode acelerar recuperação |
| **Reduz DOMS** | Moderada | Menos dor muscular pós-treino |
| **Reduz dano muscular** | Moderada | Marcadores como CK mais baixos |
| **Aumenta MPS** | Fraca-Moderada | Mais relevante para idosos |
| **Aumenta hipertrofia** | Fraca | Não comprovado em jovens saudáveis |
| **Preserva massa em imobilização** | Moderada | Útil em lesões |

### 4.5 Ratio Ômega-6:Ômega-3

**Problema moderno:**
- Dieta ocidental: ratio 15-20:1 (ômega-6:ômega-3)
- Ideal estimado: 4:1 ou menos
- Excesso de ômega-6 promove inflamação

**Soluções:**
1. Aumentar ômega-3 (peixes, suplementos)
2. Reduzir ômega-6 (menos óleos vegetais refinados)
3. Preferir azeite (ômega-9) a óleos de soja/milho

## 5. GORDURA SATURADA

### 5.1 Desmistificando a Gordura Saturada

```
GORDURA SATURADA - A VERDADE:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  ❌ MITO ANTIGO: "Gordura saturada causa doença cardíaca"              │
│                                                                         │
│  ✅ REALIDADE ATUAL:                                                   │
│  ─────────────────                                                     │
│  • Relação é mais complexa do que se pensava                           │
│  • Contexto importa: dieta total, atividade, composição corporal       │
│  • Atletas ativos e magros toleram mais sem efeitos negativos          │
│  • Sedentários/obesos têm mais risco com excesso                       │
│  • Tipo de saturada importa (cadeia média vs longa)                    │
│                                                                         │
│  O QUE SABEMOS:                                                        │
│  ───────────────                                                       │
│  • Gordura saturada ELEVA LDL (colesterol "ruim")                      │
│  • Mas também pode elevar HDL (colesterol "bom")                       │
│  • Efeito líquido depende do indivíduo e contexto                      │
│  • Trocar por PUFA/MUFA melhora perfil lipídico                        │
│                                                                         │
│  PARA PRATICANTES DE MUSCULAÇÃO:                                       │
│  ─────────────────────────────────                                     │
│  • Gordura saturada NÃO precisa ser eliminada                          │
│  • Necessária para testosterona                                        │
│  • Manter <10% das calorias totais é prudente                          │
│  • Mas 7-10% é adequado, não precisa ir a zero                         │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 5.2 Recomendações da OMS (2023)

| Tipo de Gordura | Recomendação OMS |
|-----------------|------------------|
| **Total** | <30% das calorias |
| **Saturada** | <10% das calorias |
| **Trans** | <1% das calorias |
| **Insaturada** | Preferir como fonte principal |

### 5.3 Para Quem Treina

**Saturada para praticantes de musculação:**
- NÃO eliminar (prejudica testosterona)
- Manter em ~7-10% das calorias
- Fontes: ovos, carnes, laticínios
- Contexto: se magro e ativo, menos preocupação
- Se sobrepeso/sedentário, mais cautela

### 8.2 Gordura e Absorção de Proteína

**Mito:** "Gordura atrapalha absorção de proteína pós-treino"

**Realidade:**
- Gordura pode retardar digestão ligeiramente
- Mas NÃO reduz absorção total
- Aminoácidos ainda chegam ao músculo
- Para maioria das pessoas, não faz diferença prática
- Se preferir shake puro pós-treino, OK, mas não é obrigatório

## 9. GORDURAS E COMPOSIÇÃO CORPORAL

### 9.1 Gordura Engorda?

```
GORDURA DIETÉTICA E GORDURA CORPORAL:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  MITO: "Comer gordura = armazenar gordura"                             │
│                                                                         │
│  REALIDADE:                                                            │
│  ──────────                                                            │
│  • SUPERÁVIT CALÓRICO engorda, não gordura per se                      │
│  • Gordura é mais calórica (9 kcal/g vs 4 kcal/g)                      │
│  • Por isso é fácil exagerar em calorias comendo gordura               │
│  • Mas em balanço calórico, gordura não "engorda mais"                 │
│                                                                         │
│  ESTUDOS ISOCALÓRICOS:                                                 │
│  ─────────────────────                                                 │
│  Quando calorias são iguais:                                           │
│  • Dietas high-fat vs low-fat = mesma perda de gordura                 │
│  • O que importa é déficit calórico                                    │
│  • Tipo de macro afeta saciedade, não armazenamento                    │
│                                                                         │
│  VANTAGENS DA GORDURA:                                                 │
│  ─────────────────────                                                 │
│  • Alta saciedade por volume                                           │
│  • Retarda fome                                                        │
│  • Pode ajudar adesão à dieta                                          │
│  • Essencial para hormônios                                            │
│                                                                         │
│  DESVANTAGENS DA GORDURA:                                              │
│  ────────────────────────                                              │
│  • Muito calórica (fácil exagerar)                                     │
│  • Menos volume de comida por caloria                                  │
│  • Em superávit, gordura é facilmente armazenada                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 9.2 Low-Fat vs Low-Carb

| Aspecto | Low-Fat | Low-Carb/High-Fat |
|---------|---------|-------------------|
| **Perda de peso** | Igual (se isocalórico) | Igual (se isocalórico) |
| **Preservação muscular** | Pode ser melhor | Similar |
| **Saciedade** | Menor para alguns | Maior para alguns |
| **Performance treino** | Mantida | Pode cair inicialmente |
| **Testosterona** | Pode reduzir se muito baixo | Mantida/aumentada |
| **Adesão** | Depende da pessoa | Depende da pessoa |

**Conclusão:** Escolha baseada em preferência pessoal e adesão.

## 10. IMPLEMENTAÇÃO NO VITRÚVIO

### 10.1 Algoritmo de Recomendação de Gorduras

```typescript
interface DadosUsuario {
  peso: number
  objetivo: 'HIPERTROFIA' | 'MANUTENCAO' | 'CUTTING' | 'RECOMPOSICAO'
  caloriasDiarias: number
  preferenciaDieta: 'NORMAL' | 'LOW_CARB' | 'KETO' | 'LOW_FAT'
  consomePeixe: boolean
  frequenciaPeixeSemana: number  // 0-7
}

interface RecomendacaoGorduras {
  gramas: {min: number, max: number}
  percentualCalorias: {min: number, max: number}
  distribuicao: {
    saturada: string
    monoinsaturada: string
    poliinsaturada: string
    omega3: string
  }
  fontesPrioritarias: string[]
  alertas: string[]
}

function calcularGorduras(dados: DadosUsuario): RecomendacaoGorduras {
  let percentMin: number
  let percentMax: number
  
  // Base por objetivo e preferência
  if (dados.preferenciaDieta === 'KETO') {
    percentMin = 60
    percentMax = 75
  } else if (dados.preferenciaDieta === 'LOW_CARB') {
    percentMin = 35
    percentMax = 50
  } else if (dados.preferenciaDieta === 'LOW_FAT') {
    percentMin = 20
    percentMax = 25
  } else {
    // Normal
    switch (dados.objetivo) {
      case 'HIPERTROFIA':
        percentMin = 20
        percentMax = 30
        break
      case 'CUTTING':
        percentMin = 20
        percentMax = 30
        break
      default:
        percentMin = 25
        percentMax = 35
    }
  }
  
  const gramasMin = Math.round((dados.caloriasDiarias * (percentMin / 100)) / 9)
  const gramasMax = Math.round((dados.caloriasDiarias * (percentMax / 100)) / 9)
  
  // Verificar mínimo absoluto
  const minimoAbsoluto = Math.round(dados.peso * 0.5)
  const gramasMinFinal = Math.max(gramasMin, minimoAbsoluto)
  
  // Distribuição
  const distribuicao = {
    saturada: '<10% das calorias (~' + Math.round(dados.caloriasDiarias * 0.08 / 9) + 'g)',
    monoinsaturada: '10-15% das calorias',
    poliinsaturada: '5-10% das calorias',
    omega3: dados.consomePeixe && dados.frequenciaPeixeSemana >= 2 
      ? 'Adequado via dieta' 
      : '1-2g EPA+DHA/dia (suplementar)'
  }
  
  // Fontes prioritárias
  const fontes = [
    'Azeite de oliva extra virgem',
    'Abacate',
    'Nozes e amêndoas',
    'Ovos inteiros'
  ]
  
  if (dados.consomePeixe) {
    fontes.splice(2, 0, 'Peixes gordos (salmão, sardinha)')
  }
  
  // Alertas
  const alertas: string[] = []
  
  if (dados.preferenciaDieta === 'LOW_FAT' && dados.objetivo === 'HIPERTROFIA') {
    alertas.push('Low-fat pode reduzir testosterona. Monitore energia e libido.')
  }
  
  if (!dados.consomePeixe && dados.frequenciaPeixeSemana < 2) {
    alertas.push('Considere suplementar ômega-3 (EPA+DHA) para recuperação ótima.')
  }
  
  if (dados.preferenciaDieta === 'KETO' && dados.objetivo === 'HIPERTROFIA') {
    alertas.push('Keto com alto volume de gordura pode dificultar superávit.')
  }
  
  return {
    gramas: {min: gramasMinFinal, max: gramasMax},
    percentualCalorias: {min: percentMin, max: percentMax},
    distribuicao,
    fontesPrioritarias: fontes,
    alertas
  }
}
```

## 11. REGRAS DE OURO - GORDURAS

```
REGRAS DO VITRÚVIO PARA GORDURAS:

1. NÃO IR MUITO BAIXO
   • Mínimo: 20% das calorias ou 0.5 g/kg
   • Low-fat extremo pode reduzir testosterona
   • Gordura é essencial, não opcional

2. PRIORIZAR QUALIDADE
   • Monoinsaturadas (azeite, abacate): BASE
   • Ômega-3 (peixes, suplemento): ESSENCIAL
   • Saturadas (ovos, carnes): MODERAR
   • Trans industrial: EVITAR

3. ÔMEGA-3 PARA RECUPERAÇÃO
   • 1-2g EPA+DHA/dia para atletas
   • Peixes gordos 2-3x/semana OU suplementar
   • Reduz inflamação e pode ajudar recuperação

4. SATURADA NÃO É VILÃ
   • Necessária para testosterona
   • Manter <10% das calorias
   • Não eliminar completamente

5. TRANS INDUSTRIAL É VILÃ
   • Evitar ao máximo
   • Checar rótulos: "gordura parcialmente hidrogenada"
   • Frituras e processados são fontes comuns

6. DISTRIBUIR AO LONGO DO DIA
   • ~20-30% da gordura por refeição principal
   • Evitar muita gordura pré-treino imediato
   • Com vitaminas lipossolúveis (A, D, E, K)

7. TOTAL CALÓRICO IMPORTA MAIS
   • Gordura não engorda - superávit engorda
   • Gordura é densa (9 kcal/g) - fácil exagerar
   • Em déficit, gordura não é problema

8. INDIVIDUALIZAR
   • Alguns preferem mais gordura (saciedade)
   • Outros preferem mais carbs (energia)
   • Adesão > perfeição teórica
```
