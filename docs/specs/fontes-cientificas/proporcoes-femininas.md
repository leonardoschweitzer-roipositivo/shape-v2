## 1. VISÃO GERAL

### 1.1 O que é esta SPEC?

Esta SPEC contém o conhecimento científico sobre **proporções corporais femininas ideais** que o VITRÚVIO deve usar para:

- Avaliar proporções de atletas mulheres
- Calcular metas de desenvolvimento muscular
- Identificar pontos fracos e fortes
- Criar planos de treino priorizados
- Adaptar o conceito de Golden Ratio para o corpo feminino

### 1.2 Diferenças Fundamentais: Homens vs Mulheres

| Aspecto | Homens | Mulheres |
|---------|--------|----------|
| **Proporção chave** | Ombros/Cintura (V-taper) | Cintura/Quadril (Ampulheta) |
| **Índice principal** | Adônis (ombros÷cintura) | WHR (cintura÷quadril) |
| **Foco estético** | Largura de ombros | Curvas e cintura fina |
| **Distribuição de gordura** | Abdominal | Quadril/Coxas/Glúteos |
| **% gordura ideal atlético** | 8-15% | 15-22% |
| **% gordura essencial** | 2-5% | 10-13% |

### 1.3 Por que Proporções Diferentes?

As proporções ideais femininas diferem por razões biológicas e evolutivas:

1. **Dimorfismo sexual:** Hormônios (estrogênio vs testosterona) criam padrões diferentes de deposição de gordura e desenvolvimento muscular

2. **Sinalização reprodutiva:** WHR baixo (0.7) está correlacionado com fertilidade e saúde

3. **Distribuição de gordura:** Mulheres naturalmente depositam mais gordura em quadril, coxas e glúteos (padrão ginóide)

4. **Capacidade muscular:** Mulheres têm ~60-70% da capacidade de ganho muscular absoluto dos homens, mas ganhos proporcionais similares

## 2. A CIÊNCIA DO WHR (WAIST-TO-HIP RATIO)

### 2.1 O que é WHR?

```
WHR = Circunferência da Cintura ÷ Circunferência do Quadril

Exemplo:
Cintura: 65 cm
Quadril: 95 cm
WHR = 65 ÷ 95 = 0.68
```

### 2.2 Pesquisas sobre WHR e Atratividade

**Estudo clássico: Singh (1993)**
- WHR de 0.7 consistentemente avaliado como mais atraente
- Correlação com percepção de saúde, fertilidade e juventude
- Replicado em múltiplas culturas (EUA, Alemanha, Índia, etc.)

**Meta-análise de preferências:**

| WHR | Avaliação |
|-----|-----------|
| 0.6 | Atraente (preferido em algumas culturas asiáticas) |
| **0.7** | **Mais atraente universalmente** |
| 0.8 | Moderadamente atraente |
| 0.9 | Menos atraente |
| 1.0+ | Associado a riscos de saúde |

**Estudo Hübner & Ufken (2024):**
- Curvatura corporal pode ser melhor preditor que WHR isolado
- A "forma" geral importa mais que números específicos
- WHR de 0.7 ainda emergiu como preferido

### 2.3 WHR e Saúde

| WHR | Classificação (Mulheres) | Risco de Saúde |
|-----|--------------------------|----------------|
| < 0.75 | Baixo risco | Excelente |
| 0.75-0.80 | Risco moderado | Bom |
| 0.80-0.85 | Risco elevado | Atenção |
| > 0.85 | Risco alto | Preocupante (OMS) |

**A OMS define WHR > 0.85 como indicador de obesidade abdominal em mulheres.**

### 3.1 Sistema de Proporções (Venus Index)

John Barban propôs o "Venus Index" como equivalente feminino do Índice de Adônis:

```
PROPORÇÕES VENUS INDEX:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  BASE: Cintura (referência para todos os cálculos)                     │
│                                                                         │
│  PROPORÇÕES IDEAIS:                                                    │
│  ─────────────────────                                                 │
│  Cintura = Altura × 0.38 a 0.40                                        │
│  Quadril = Cintura × 1.42 (WHR ≈ 0.7)                                  │
│  Ombros = Cintura × 1.45                                               │
│  Busto ≈ Quadril (similar ou levemente menor)                          │
│                                                                         │
│  EXEMPLO (mulher de 1.65m):                                            │
│  ─────────────────────────────                                         │
│  Altura: 165 cm                                                        │

│  Quadril ideal: 63 × 1.42 = 89.5 cm (~90 cm)                           │
│  Ombros ideal: 63 × 1.45 = 91.4 cm (~91 cm)                            │

│                                                                         │

```

## 6. CALCULANDO PROPORÇÕES FEMININAS

### 6.1 Fórmulas do VITRÚVIO

```typescript
interface MedidasFemininas {
  altura: number           // cm
  cintura: number          // cm
  quadril: number          // cm
  busto: number            // cm
  ombros: number           // cm
  coxaEsquerda: number     // cm
  coxaDireita: number      // cm
  bracoEsquerdo: number    // cm
  bracoDireito: number     // cm
  panturrilhaEsquerda: number  // cm
  panturrilhaDireita: number   // cm
}

interface ProporcoesIdeaisFemininas {
  cintura: number
  quadril: number
  ombros: number
  busto: number
  coxa: number
  braco: number
  panturrilha: number
  whr: number
}

interface AnaliseProporcoesF {
  whr: number
  whrIdeal: number
  whrStatus: 'EXCELENTE' | 'BOM' | 'MEDIO' | 'PRECISA_MELHORAR'
  
  proporcoesAtuais: Record<string, number>
  proporcoesIdeais: Record<string, number>
  diferencaPercentual: Record<string, number>
  
  pontosFortes: string[]
  pontosFracos: string[]
  prioridades: string[]
  
  tipoCorpo: 'AMPULHETA' | 'PERA' | 'RETANGULO' | 'MACA'
  
  recomendacoes: string[]
}

function calcularProporcoesIdeaisFemininas(altura: number): ProporcoesIdeaisFemininas {
  // Cintura ideal: 38-40% da altura
  const cinturaIdeal = altura * 0.39
  
  // WHR ideal: 0.7
  const whrIdeal = 0.70
  
  // Quadril: para atingir WHR 0.7
  const quadrilIdeal = cinturaIdeal / whrIdeal  // = cintura × 1.43
  
  // Ombros: levemente maiores que quadril para X-shape
  const ombrosIdeal = cinturaIdeal * 1.45
  
  // Busto: similar ao quadril
  const bustoIdeal = quadrilIdeal * 0.98
  
  // Coxa: 60-65% da cintura
  const coxaIdeal = cinturaIdeal * 0.62
  
  // Braço: 30-35% da cintura
  const bracoIdeal = cinturaIdeal * 0.32
  
  // Panturrilha: ~60% da coxa
  const panturrilhaIdeal = coxaIdeal * 0.62
  
  return {
    cintura: Math.round(cinturaIdeal * 10) / 10,
    quadril: Math.round(quadrilIdeal * 10) / 10,
    ombros: Math.round(ombrosIdeal * 10) / 10,
    busto: Math.round(bustoIdeal * 10) / 10,
    coxa: Math.round(coxaIdeal * 10) / 10,
    braco: Math.round(bracoIdeal * 10) / 10,
    panturrilha: Math.round(panturrilhaIdeal * 10) / 10,
    whr: whrIdeal
  }
}

function calcularWHR(cintura: number, quadril: number): number {
  return Math.round((cintura / quadril) * 100) / 100
}

function classificarWHR(whr: number): string {
  if (whr <= 0.70) return 'EXCELENTE'
  if (whr <= 0.75) return 'BOM'
  if (whr <= 0.80) return 'MEDIO'
  if (whr <= 0.85) return 'PRECISA_MELHORAR'
  return 'RISCO_SAUDE'
}

function determinarTipoCorpo(
  busto: number, 
  cintura: number, 
  quadril: number
): string {
  const diffBustoQuadril = Math.abs(busto - quadril)
  const diffBustoCintura = busto - cintura
  const diffQuadrilCintura = quadril - cintura
  
  // Ampulheta: busto ≈ quadril E cintura bem menor
  if (diffBustoQuadril <= 2.5 && diffBustoCintura >= 23 && diffQuadrilCintura >= 25) {
    return 'AMPULHETA'
  }
  
  // Pera: quadril > busto significativamente
  if ((quadril - busto) >= 9) {
    return 'PERA'
  }
  
  // Maçã/Triângulo invertido: busto > quadril significativamente
  if ((busto - quadril) >= 9) {
    return 'MACA'
  }
  
  // Retângulo: medidas similares, pouca definição de cintura
  if (diffBustoCintura < 23 && diffQuadrilCintura < 25) {
    return 'RETANGULO'
  }
  
  // Default
  return 'RETANGULO'
}
```

**Para desenvolver ombros (criar largura):**

**Para desenvolver glúteos (criar curvas):**
- Hip thrust (principal)
- Bulgarian split squat
- Stiff e variações
- Abdução (máquina e banda)

- Frog pumps

- EVITAR oblíquos pesados (engrossam)

```typescript
interface AvaliacaoFeminina {
  medidas: MedidasFemininas
  idade: number
  experiencia: 'INICIANTE' | 'INTERMEDIARIO' | 'AVANCADO'
  objetivo: 'ESTETICA' | 'BIKINI' | 'WELLNESS' | 'FIGURE' | 'SAUDE'
  faseDoCliclo?: number  // dia do ciclo, se disponível
}

interface ResultadoAvaliacaoF {
  proporcoesAtuais: Record<string, number>
  proporcoesIdeais: Record<string, number>
  
  whr: number
  whrClassificacao: string
  
  tipoCorpo: string
  
  ffmi?: number
  ffmiClassificacao?: string
  
  pontosFortes: Array<{grupo: string, motivo: string}>
  pontosFracos: Array<{grupo: string, diferenca: number}>
  
  prioridadesTreino: Array<{
    grupo: string
    volume: string
    frequencia: number
    justificativa: string
  }>
  
  metasCircunferencia: Record<string, {atual: number, meta: number, diferenca: number}>
  
  alertas: string[]
}

function avaliarProporoesFemininas(dados: AvaliacaoFeminina): ResultadoAvaliacaoF {
  const { medidas } = dados
  
  // Calcular proporções ideais baseadas na altura
  const ideais = calcularProporcoesIdeaisFemininas(medidas.altura)
  
  // Calcular WHR
  const whr = calcularWHR(medidas.cintura, medidas.quadril)
  const whrClass = classificarWHR(whr)
  
  // Determinar tipo de corpo
  const tipoCorpo = determinarTipoCorpo(
    medidas.busto, 
    medidas.cintura, 
    medidas.quadril
  )
  
  // Identificar pontos fracos (mais de 5% acima ou abaixo do ideal)
  const pontosFracos: Array<{grupo: string, diferenca: number}> = []
  const pontosFortes: Array<{grupo: string, motivo: string}> = []
  
  // Análise de cintura
  const diffCintura = ((medidas.cintura - ideais.cintura) / ideais.cintura) * 100
  if (diffCintura > 5) {
    pontosFracos.push({grupo: 'Cintura', diferenca: diffCintura})
  } else if (diffCintura <= 0) {
    pontosFortes.push({grupo: 'Cintura', motivo: 'No ideal ou abaixo'})
  }
  
  // Análise de ombros
  const diffOmbros = ((medidas.ombros - ideais.ombros) / ideais.ombros) * 100
  if (diffOmbros < -5) {
    pontosFracos.push({grupo: 'Ombros', diferenca: diffOmbros})
  } else if (diffOmbros >= 0) {
    pontosFortes.push({grupo: 'Ombros', motivo: 'Bem desenvolvidos'})
  }
  
  // Análise de glúteos/quadril
  const diffQuadril = ((medidas.quadril - ideais.quadril) / ideais.quadril) * 100
  if (diffQuadril < -5 && dados.objetivo !== 'SAUDE') {
    pontosFracos.push({grupo: 'Glúteos/Quadril', diferenca: diffQuadril})
  }
  
  // Gerar prioridades de treino baseadas no tipo de corpo e pontos fracos
  const prioridades = gerarPrioridadesTreino(tipoCorpo, pontosFracos, dados.objetivo)
  
  // Alertas
  const alertas: string[] = []
  if (whr > 0.85) {
    alertas.push('WHR acima de 0.85 indica risco aumentado de saúde. Recomendamos foco em redução de cintura.')
  }
  if (dados.faseDoCliclo && dados.faseDoCliclo >= 15 && dados.faseDoCliclo <= 28) {
    alertas.push('Você está na fase lútea. Medidas podem estar 1-3cm maiores que o normal devido à retenção de água.')
  }
  
  return {
    proporcoesAtuais: {
      cintura: medidas.cintura,
      quadril: medidas.quadril,
      ombros: medidas.ombros,
      busto: medidas.busto
    },
    proporcoesIdeais: ideais,
    whr,
    whrClassificacao: whrClass,
    tipoCorpo,
    pontosFortes,
    pontosFracos,
    prioridadesTreino: prioridades,
    metasCircunferencia: {
      cintura: {atual: medidas.cintura, meta: ideais.cintura, diferenca: medidas.cintura - ideais.cintura},
      quadril: {atual: medidas.quadril, meta: ideais.quadril, diferenca: medidas.quadril - ideais.quadril},
      ombros: {atual: medidas.ombros, meta: ideais.ombros, diferenca: medidas.ombros - ideais.ombros}
    },
    alertas
  }
}

function gerarPrioridadesTreino(
  tipoCorpo: string, 
  pontosFracos: Array<{grupo: string, diferenca: number}>,
  objetivo: string
): Array<{grupo: string, volume: string, frequencia: number, justificativa: string}> {
  
  const prioridades = []
  
  // Regras por tipo de corpo
  if (tipoCorpo === 'RETANGULO') {
    prioridades.push({
      grupo: 'Glúteos',
      volume: 'ALTO (16-20 séries/sem)',
      frequencia: 3,
      justificativa: 'Criar curvas no corpo retangular'
    })
    prioridades.push({
      grupo: 'Deltoides',
      volume: 'ALTO (14-18 séries/sem)',
      frequencia: 3,
      justificativa: 'Ampliar ombros para criar X-shape'
    })
  }
  
  if (tipoCorpo === 'PERA') {
    prioridades.push({
      grupo: 'Deltoides',
      volume: 'ALTO (16-20 séries/sem)',
      frequencia: 3,
      justificativa: 'Equilibrar proporções superior/inferior'
    })
    prioridades.push({
      grupo: 'Costas (largura)',
      volume: 'ALTO (14-18 séries/sem)',
      frequencia: 2,
      justificativa: 'Criar V-taper para balancear quadril'
    })
  }
  
  if (tipoCorpo === 'MACA') {
    prioridades.push({
      grupo: 'Glúteos',
      volume: 'ALTO (16-20 séries/sem)',
      frequencia: 3,
      justificativa: 'Desenvolver inferior para equilibrar'
    })
    prioridades.push({
      grupo: 'Quadríceps/Isquiotibiais',
      volume: 'ALTO (14-16 séries/sem)',
      frequencia: 2,
      justificativa: 'Adicionar volume às pernas'
    })
    // Nota: também precisa reduzir cintura via déficit
  }
  
  // Ajustes por objetivo
  if (objetivo === 'WELLNESS') {
    // Wellness prioriza inferior extremamente
    const gluteoIndex = prioridades.findIndex(p => p.grupo === 'Glúteos')
    if (gluteoIndex >= 0) {
      prioridades[gluteoIndex].volume = 'MUITO ALTO (20-24 séries/sem)'
    } else {
      prioridades.unshift({
        grupo: 'Glúteos',
        volume: 'MUITO ALTO (20-24 séries/sem)',
        frequencia: 3,
        justificativa: 'Wellness exige glúteos/coxas muito desenvolvidos'
      })
    }
  }
  
  return prioridades
}

```
REGRAS DO VITRÚVIO PARA MULHERES:

1. WHR 0.70 É O ALVO UNIVERSAL
   • Correlacionado com saúde e estética
   • Funciona reduzindo cintura E/OU aumentando quadril

2. CINTURA É A BASE DE TUDO
   • Todas as proporções derivam da cintura
   • Reduzir cintura melhora TODOS os índices
   • Principal ferramenta: déficit calórico

3. DESENVOLVER OMBROS CRIA "ILUSÃO" DE CINTURA MENOR
   • Ombros largos fazem cintura parecer menor
   • X-shape é mais importante que tamanho absoluto

4. GLÚTEOS SÃO PRIORIDADE PARA MAIORIA DAS MULHERES
   • Criam curvas
   • Melhoram WHR
   • Podem (e devem) ser treinados 3x/semana

5. EVITAR OBLÍQUOS PESADOS
   • Hipertrofiam cintura
   • Fazer apenas reto abdominal
   • Vacuum é permitido e recomendado

6. TIPO DE CORPO DETERMINA ESTRATÉGIA
   • Retângulo: criar curvas (glúteos + ombros)
   • Pera: desenvolver superior
   • Maçã: reduzir cintura + desenvolver inferior
   • Ampulheta: manter proporções

7. FASE DO CICLO AFETA MEDIDAS
   • Medir sempre na mesma fase
   • Fase folicular (dias 5-10) é ideal
   • Esperar 1-3 cm de variação na fase lútea

8. % GORDURA FEMININO É NATURALMENTE MAIOR
   • Essencial: 10-13% (não sustentável)
   • Atlético: 15-20%
   • Saudável: 21-28%
   • Comparar com padrões femininos, não masculinos
