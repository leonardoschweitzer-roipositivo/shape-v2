## 1. VISÃO GERAL

### 1.1 O que é esta SPEC?

Esta SPEC contém o conhecimento científico sobre **balanço energético** que o VITRÚVIO deve usar como base para:

- Definir déficit ou superávit adequado ao objetivo
- Calcular taxa de perda/ganho de peso ideal
- Preservar massa muscular em cutting
- Minimizar ganho de gordura em bulking
- Identificar quando fazer recomposição corporal

### 1.2 Conceitos Fundamentais

```
BALANÇO ENERGÉTICO:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  DÉFICIT CALÓRICO (Cutting)                                            │
│  ─────────────────────────────                                         │
│  Calorias consumidas < Calorias gastas                                 │
│  Resultado: Perda de peso (gordura + potencialmente músculo)           │
│                                                                         │
│  MANUTENÇÃO (Balanço)                                                  │
│  ────────────────────────                                              │
│  Calorias consumidas ≈ Calorias gastas                                 │
│  Resultado: Peso estável                                               │
│                                                                         │
│  SUPERÁVIT CALÓRICO (Bulking)                                          │
│  ─────────────────────────────                                         │
│  Calorias consumidas > Calorias gastas                                 │
│  Resultado: Ganho de peso (músculo + inevitavelmente alguma gordura)   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 2. DÉFICIT CALÓRICO (CUTTING)

### 2.1 Princípio Fundamental

```
REGRA DO 3.500:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  ~3.500 kcal ≈ 0.45 kg de gordura (1 libra)                            │
│                                                                         │
│  Déficit de 500 kcal/dia × 7 dias = 3.500 kcal/semana ≈ 0.5 kg        │
│                                                                         │
│  NOTA: Esta é uma aproximação útil, mas não 100% precisa.              │
│  O corpo é mais complexo (adaptação metabólica, variação de água, etc)│
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 2.2 Taxas de Perda de Peso Recomendadas

Baseado em Garthe et al. (2011) e revisões subsequentes:

```
TAXA DE PERDA DE PESO SEMANAL:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  CONSERVADORA (Melhor para preservar músculo):                         │
│  ────────────────────────────────────────────                          │
│  • 0.5% do peso corporal por semana                                    │
│  • Exemplo (80 kg): 0.4 kg/semana                                      │
│  • Déficit: ~300-400 kcal/dia                                          │
│  • Ideal para: Atletas magros (<15% BF), avançados                     │
│                                                                         │
│  MODERADA (Equilíbrio velocidade/preservação):                         │
│  ────────────────────────────────────────────                          │
│  • 0.7% do peso corporal por semana                                    │
│  • Exemplo (80 kg): 0.56 kg/semana                                     │
│  • Déficit: ~500 kcal/dia                                              │
│  • Ideal para: Maioria dos praticantes                                 │
│                                                                         │
│  AGRESSIVA (Mais rápida, maior risco):                                 │
│  ────────────────────────────────────                                  │
│  • 1.0% do peso corporal por semana                                    │
│  • Exemplo (80 kg): 0.8 kg/semana                                      │
│  • Déficit: ~750-1000 kcal/dia                                         │
│  • Ideal para: Iniciantes, pessoas com mais gordura (>25% BF)          │
│                                                                         │
│  MÁXIMA RECOMENDADA:                                                   │
│  ────────────────────                                                  │
│  • 1.4% do peso corporal por semana                                    │
│  • APENAS para obesos ou situações especiais                           │
│  • Maior risco de perda muscular                                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 2.3 Estudo Garthe et al. (2011)

**Estudo:** "Effect of two different weight-loss rates on body composition and strength and power-related performance in elite athletes"

**Metodologia:**
- 24 atletas de elite
- Grupo SR (Slow Reduction): 0.7% peso/semana
- Grupo FR (Fast Reduction): 1.4% peso/semana
- Treino de força 4x/semana

**Resultados:**

| Métrica | Slow (0.7%/sem) | Fast (1.4%/sem) |
|---------|-----------------|-----------------|
| Perda de peso total | 5.6% | 5.5% |
| Perda de gordura | 31% | 21% |
| **Massa magra** | **+2.1%** | **-0.2%** |
| Força 1RM | Aumentou | Manteve |
| Tempo de dieta | 8.5 semanas | 5.3 semanas |

**Conclusão:** Perda lenta (0.7%/sem) permitiu GANHO de massa magra durante cutting!

│  • Platô de peso > 2-3 semanas                                         │
│  • Fadiga excessiva                                                    │

│  • Fome incontrolável                                                  │

## 3. SUPERÁVIT CALÓRICO (BULKING)

### 3.1 Tipos de Bulk

```
COMPARAÇÃO DE ESTRATÉGIAS DE BULK:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  LEAN BULK (Recomendado):                                              │
│  ─────────────────────────                                             │
│  • Superávit: 200-300 kcal/dia (10-15% acima manutenção)               │
│  • Ganho de peso: 0.25-0.5 kg/semana                                   │
│  • Razão músculo:gordura melhor                                        │
│  • Cut posterior mais curto                                            │
│  • Ideal para: Intermediários e avançados                              │
│                                                                         │
│  BULK MODERADO:                                                        │
│  ─────────────────                                                     │
│  • Superávit: 300-500 kcal/dia (15-20% acima manutenção)               │
│  • Ganho de peso: 0.5-0.75 kg/semana                                   │
│  • Bom para iniciantes com metabolismo rápido                          │
│  • Ganho de gordura moderado                                           │
│                                                                         │
│  DIRTY BULK (Não recomendado):                                         │
│  ──────────────────────────────                                        │
│  • Superávit: 500-1000+ kcal/dia                                       │
│  • Ganho de peso: 1+ kg/semana                                         │
│  • Muito ganho de gordura                                              │
│  • Cut longo e difícil depois                                          │
│  • Não constrói mais músculo que lean bulk!                            │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 Estudo: Superávit Pequeno vs Grande

**Estudo:** Ribeiro et al. (2023) - "Effect of Small and Large Energy Surpluses on Strength, Muscle, and Skinfold Thickness"

**Grupos:**
- MAIN: Manutenção calórica
- MOD: Superávit de 5%
- HIGH: Superávit de 15%

**Resultados:**

| Métrica | Manutenção | 5% Superávit | 15% Superávit |
|---------|------------|--------------|---------------|
| Ganho de músculo | Similar | Similar | Similar |
| Ganho de gordura | Baixo | Moderado | **Alto** |
| Força | Aumentou | Aumentou | Aumentou |

**Conclusão:** Superávit maior (15%) não construiu mais músculo, apenas mais gordura!

## 4. RECOMPOSIÇÃO CORPORAL

### 4.1 O que é Recomposição?

```
RECOMPOSIÇÃO CORPORAL:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  DEFINIÇÃO:                                                            │
│  Ganhar músculo E perder gordura SIMULTANEAMENTE                       │
│                                                                         │
│  COMO É POSSÍVEL?                                                      │
│  • Músculo e gordura são "contas bancárias" separadas                  │
│  • Pode depositar em uma enquanto retira da outra                      │
│  • Desde que haja estímulo (treino) + substrato (proteína)             │
│                                                                         │
│  BALANÇO ENERGÉTICO:                                                   │
│  • Pode estar em leve déficit LÍQUIDO                                  │
│  • Energia do músculo construído (~3.600 kcal/kg) vs                   │
│    Energia da gordura perdida (~7.700 kcal/kg)                         │
│  • Se ganhou 1kg músculo e perdeu 1kg gordura:                         │
│    Déficit líquido de ~4.100 kcal (muito pequeno por dia)              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4.2 Quem Pode Fazer Recomposição?

```
CANDIDATOS IDEAIS PARA RECOMPOSIÇÃO:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  ALTA PROBABILIDADE DE SUCESSO:                                        │
│  ─────────────────────────────────                                     │
│  ✓ Iniciantes em treino de força (primeiros 1-2 anos)                  │
│  ✓ Pessoas retornando após pausa longa (muscle memory)                 │
│  ✓ Pessoas com alto % de gordura (>20% homens, >30% mulheres)         │
│  ✓ Usuários de substâncias anabólicas (TRT, etc)                       │
│  ✓ Pessoas subtreinadas (treinavam mal antes)                          │
│                                                                         │
│  BAIXA PROBABILIDADE DE SUCESSO:                                       │
│  ─────────────────────────────────                                     │
│  ✗ Atletas avançados/elite já magros                                   │
│  ✗ Pessoas já com baixo % de gordura (<12% homens, <20% mulheres)     │
│  ✗ Pessoas já perto do potencial genético                              │
│                                                                         │
│  PARA ESSES GRUPOS: Ciclar entre bulk e cut é mais eficiente!          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 5. ADAPTAÇÃO METABÓLICA

### 5.1 O que é Adaptação Metabólica?

```
METABOLIC ADAPTATION (Adaptive Thermogenesis):

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  DEFINIÇÃO:                                                            │
│  Redução do gasto energético ALÉM do esperado pela perda de peso       │
│                                                                         │
│  COMPONENTES:                                                          │
│  ────────────                                                          │
│  1. Redução da TMB (5-15%)                                             │
│     • Hormônios tireoidianos diminuem                                  │
│     • Eficiência metabólica aumenta                                    │
│                                                                         │
│  2. Redução do NEAT (maior componente!)                                │
│     • Movimentos inconscientes diminuem                                │
│     • Pessoa se move menos sem perceber                                │
│     • Pode representar 300-500 kcal/dia de redução                     │
│                                                                         │
│  3. Redução do EAT                                                     │
│     • Menos energia = treinos menos intensos                           │
│     • Performance diminui                                              │
│                                                                         │
│  4. TEF permanece relativamente estável                                │
│     • Mas absoluto menor (menos comida)                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

```
QUANTO O METABOLISMO ADAPTA:

│                                                                         │

│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 7. COMO O VITRÚVIO DEVE CALCULAR

### 7.1 Algoritmo de Prescrição

```typescript
interface ParametrosBalanco {
  peso: number              // kg
  gorduraCorporal: number   // %
  tdee: number              // kcal (calculado previamente)
  objetivo: 'CUT' | 'MANUTENCAO' | 'BULK' | 'RECOMP'
  experiencia: 'INICIANTE' | 'INTERMEDIARIO' | 'AVANCADO' | 'ELITE'
  urgencia: 'CONSERVADOR' | 'MODERADO' | 'AGRESSIVO'
  semanasEmDieta: number    // para ajustar por adaptação
}

interface PlanoEnergetico {
  caloriasDiarias: number
  deficitOuSuperavit: number
  perdaOuGanhoPorSemana: number  // kg
  duracaoEstimada: number        // semanas
  proteinaMinima: number         // g/dia
  ajustesRecomendados: string[]
}

function calcularBalancoEnergetico(params: ParametrosBalanco): PlanoEnergetico {
  let ajuste = 0
  let taxaSemanal = 0
  
  if (params.objetivo === 'CUT') {
    // Determinar taxa de perda baseada em BF% e urgência
    if (params.gorduraCorporal > 25) {
      // Alto BF - pode ser mais agressivo
      switch (params.urgencia) {
        case 'CONSERVADOR': taxaSemanal = 0.007; break  // 0.7%
        case 'MODERADO': taxaSemanal = 0.01; break      // 1.0%
        case 'AGRESSIVO': taxaSemanal = 0.012; break    // 1.2%
      }
    } else if (params.gorduraCorporal > 15) {
      // BF médio
      switch (params.urgencia) {
        case 'CONSERVADOR': taxaSemanal = 0.005; break  // 0.5%
        case 'MODERADO': taxaSemanal = 0.007; break     // 0.7%
        case 'AGRESSIVO': taxaSemanal = 0.01; break     // 1.0%
      }
    } else {
      // BF baixo - deve ser conservador
      switch (params.urgencia) {
        case 'CONSERVADOR': taxaSemanal = 0.003; break  // 0.3%
        case 'MODERADO': taxaSemanal = 0.005; break     // 0.5%
        case 'AGRESSIVO': taxaSemanal = 0.007; break    // 0.7%
      }
    }
    
    const perdaSemanal = params.peso * taxaSemanal
    // ~7700 kcal por kg de gordura
    ajuste = -(perdaSemanal * 7700) / 7
    
  } else if (params.objetivo === 'BULK') {
    // Superávit baseado em experiência
    switch (params.experiencia) {
      case 'INICIANTE':
        ajuste = params.urgencia === 'AGRESSIVO' ? 500 : 350
        break
      case 'INTERMEDIARIO':
        ajuste = params.urgencia === 'AGRESSIVO' ? 300 : 200
        break
      case 'AVANCADO':
        ajuste = params.urgencia === 'AGRESSIVO' ? 200 : 150
        break
      case 'ELITE':
        ajuste = 100
        break
    }
  } else if (params.objetivo === 'RECOMP') {
    // Manutenção ou leve déficit
    ajuste = -150
  }
  
  // Ajustar por adaptação metabólica em déficits prolongados
  let tdeeAjustado = params.tdee
  if (params.objetivo === 'CUT' && params.semanasEmDieta > 8) {
    const adaptacao = Math.min(0.15, 0.05 + params.semanasEmDieta * 0.005)
    tdeeAjustado = params.tdee * (1 - adaptacao)
  }
  
  const caloriasDiarias = Math.round(tdeeAjustado + ajuste)
  
  // Calcular proteína
  let proteinaKg = 2.0
  if (params.objetivo === 'CUT') proteinaKg = 2.4
  if (params.gorduraCorporal > 25) proteinaKg = 2.0  // obeso, calcular sobre peso ajustado
  const proteinaMinima = Math.round(params.peso * proteinaKg)
  
  // Calcular duração estimada (para cut)
  let duracaoEstimada = 0
  if (params.objetivo === 'CUT') {
    const gorduraAtual = params.peso * params.gorduraCorporal / 100
    const pesoMeta = params.gorduraCorporal > 20 
      ? params.peso * 0.90  // perder 10% do peso
      : params.peso * 0.95  // perder 5% do peso
    const aPerder = params.peso - pesoMeta
    const perdaSemanal = params.peso * taxaSemanal
    duracaoEstimada = Math.ceil(aPerder / perdaSemanal)
  }
  
  // Preparar recomendações
  const ajustesRecomendados: string[] = []
  
  if (params.objetivo === 'CUT') {
    ajustesRecomendados.push(`Meta: perder ${(taxaSemanal * 100).toFixed(1)}% do peso por semana`)
    if (params.semanasEmDieta > 8) {
      ajustesRecomendados.push('Considere um diet break de 1-2 semanas em manutenção')
    }
    if (params.gorduraCorporal < 12) {
      ajustesRecomendados.push('BF muito baixo - monitore energia e performance cuidadosamente')
    }
  } else if (params.objetivo === 'BULK') {
    ajustesRecomendados.push(`Meta: ganhar ~${(ajuste * 7 / 7700).toFixed(2)} kg por semana`)
    if (params.gorduraCorporal > 18) {
      ajustesRecomendados.push('Considere iniciar com mini-cut antes do bulk')
    }
  }
  
  return {
    caloriasDiarias,
    deficitOuSuperavit: ajuste,
    perdaOuGanhoPorSemana: Math.abs(ajuste * 7 / 7700),
    duracaoEstimada,
    proteinaMinima,
    ajustesRecomendados
  }
}
```

### 7.2 Exemplo de Aplicação

```
ATLETA: Leonardo
─────────────────────────────────────────────────────
Peso: 82.5 kg
Gordura corporal: 15.8%
TDEE: 2.667 kcal/dia
Objetivo: CUT (cutting)
Experiência: ELITE (16 anos)
Urgência: MODERADA
Semanas em dieta: 0 (início)

CÁLCULO:
─────────────────────────────────────────────────────

BF% = 15.8% (médio-baixo) → Taxa conservadora-moderada

Taxa semanal = 0.5% do peso corporal
Perda semanal = 82.5 × 0.005 = 0.41 kg/semana

Déficit calórico = 0.41 × 7700 / 7 = 451 kcal/dia

Calorias diárias = 2.667 - 451 = 2.216 kcal/dia

Proteína = 82.5 × 2.4 = 198g/dia (alta, para preservar músculo)

RESULTADO:
─────────────────────────────────────────────────────
• Calorias diárias: 2.200 kcal
• Déficit: ~450 kcal/dia
• Perda esperada: ~0.4 kg/semana
• Proteína mínima: 200g/dia
• Tempo estimado (para chegar a 12% BF): ~8 semanas
```

### 8.3 Regras de Ouro

```
REGRAS DO VITRÚVIO PARA BALANÇO ENERGÉTICO:

1. QUANTO MAIS MAGRO, MAIS CONSERVADOR O DÉFICIT
   • Alto BF: pode usar déficit agressivo
   • Baixo BF: déficit pequeno para preservar músculo

2. MAIS SUPERÁVIT ≠ MAIS MÚSCULO
   • Corpo tem limite de síntese proteica
   • Excesso vira gordura, não músculo

3. AJUSTES PEQUENOS E FREQUENTES
   • 100-200 kcal de cada vez
   • Avaliar a cada 2-3 semanas

4. PROTEÍNA SEMPRE ALTA
   • Especialmente em déficit (2.2-3.0 g/kg)
   • É o pilar da preservação muscular

5. TREINO DE FORÇA É INEGOCIÁVEL
   • Essencial tanto no cut quanto no bulk
   • Manter intensidade (carga), ajustar volume se necessário

6. PESO NA BALANÇA É APENAS UM INDICADOR
   • Usar também: medidas, fotos, força no treino
   • Flutuações são normais

7. RECOMPOSIÇÃO É POSSÍVEL (para alguns)
   • Iniciantes, detrained, alto BF
   • Não espere isso de atletas avançados
```
