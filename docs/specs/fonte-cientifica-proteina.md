## 1. VISÃO GERAL

### 1.1 O que é esta SPEC?

Esta SPEC contém o conhecimento científico sobre **ingestão de proteína** que o VITRÚVIO deve usar como base para:

- Calcular necessidades proteicas diárias
- Distribuir proteína entre refeições
- Ajustar para diferentes objetivos (bulk, cut, manutenção)
- Recomendar fontes e timing de proteína

### 1.2 Definição

> **Proteína** é um macronutriente essencial composto de aminoácidos, fundamental para a síntese proteica muscular (MPS), recuperação e crescimento muscular.

### 2.2 Meta-Análise de Morton et al. (2018)

**Estudo:** "A systematic review, meta-analysis and meta-regression of the effect of protein supplementation on resistance training-induced gains in muscle mass and strength"

**Metodologia:**
- 49 estudos randomizados controlados
- 1.863 participantes
- Análise de regressão de dois segmentos

**Resultados principais:**

| Métrica | Resultado |
|---------|-----------|
| Breakpoint identificado | 1.62 g/kg/dia |
| Intervalo de confiança (95%) | 1.03 - 2.20 g/kg/dia |
| Significância do breakpoint | p = 0.079 (não significativo) |
| Ganho adicional FFM | +0.30 kg com suplementação |

**Conclusão original:** Acima de ~1.6 g/kg/dia, benefícios adicionais são mínimos.

### 2.3 Reanálise Crítica (Stronger by Science, 2025)

O pesquisador Greg Nuckols reanalisou os dados e identificou limitações:

```
PROBLEMAS COM O BREAKPOINT DE 1.6 g/kg:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  1. SIGNIFICÂNCIA ESTATÍSTICA                                          │
│     • p = 0.079 (não significativo a 0.05)                             │
│     • IC muito amplo (1.03-2.20 g/kg)                                  │
│                                                                         │
│  2. POPULAÇÃO ESTUDADA                                                 │
│     • 39 de 49 estudos usaram indivíduos NÃO TREINADOS                │
│     • Estudos com treinados tinham ingestão já alta                    │
│                                                                         │
│  3. RANGE DE INGESTÃO                                                  │
│     • Maioria dos estudos: 0.9-2.25 g/kg                               │
│     • Poucos dados acima de 2.0 g/kg                                   │
│                                                                         │
│  4. SIMPSON'S PARADOX                                                  │
│     • Correlação negativa entre baseline e ganhos                      │
│     • Distorce o breakpoint aparente                                   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

BREAKPOINT REAVALIADO:

• Mais provável: ~2.0 g/kg/dia
• Range plausível: 1.7-2.35 g/kg/dia
• Recomendação atualizada: 2.0-2.35 g/kg/dia para atletas treinados
```

### 2.4 Meta-Análise de Nunes et al. (2022)

**Estudo:** "Systematic review and meta-analysis of protein intake to support muscle mass and function in healthy adults"

**Metodologia:**
- 74 estudos randomizados controlados
- Análise por faixa etária e nível de atividade

**Resultados:**

| Grupo | Ingestão Efetiva |
|-------|-----------------|
| Jovens (<65 anos) + treino | ≥1.6 g/kg/dia |
| Idosos (≥65 anos) + treino | 1.2-1.59 g/kg/dia já mostra efeito |
| Sem treino de força | Efeitos inconsistentes |

**Conclusão:** O treino de resistência é essencial para que a proteína adicional resulte em ganhos de massa magra.

## 3. LEUCINA E O THRESHOLD ANABÓLICO

### 3.1 O Papel da Leucina

> **Leucina** é o aminoácido de cadeia ramificada (BCAA) que atua como "gatilho" da síntese proteica muscular através da ativação da via mTORC1.

```
MECANISMO DA LEUCINA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  Ingestão de proteína                                                  │
│         │                                                               │
│         ▼                                                               │
│  Digestão → Aminoácidos no sangue                                      │
│         │                                                               │
│         ▼                                                               │
│  Leucina entra na célula muscular                                      │
│         │                                                               │
│         ▼                                                               │
│  Leucina ATIVA via mTORC1/P70S6K                                       │
│         │                                                               │
│         ▼                                                               │
│  INICIA síntese proteica muscular (MPS)                                │
│         │                                                               │
│         ▼                                                               │
│  MPS permanece elevada por ~2-3 horas                                  │
│         │                                                               │
│         ▼                                                               │
│  Retorna ao baseline (independente de leucina ainda alta)              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 Leucine Threshold (Limiar de Leucina)

```
QUANTIDADE DE LEUCINA PARA ATIVAR MPS:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  JOVENS (<40 anos):                                                    │
│  • Threshold: ~2.5 g leucina por refeição                              │
│  • Equivale a: ~20-25 g de proteína de alta qualidade                  │
│                                                                         │
│  IDOSOS (>60 anos):                                                    │
│  • Threshold: ~3-4 g leucina por refeição                              │
│  • Equivale a: ~30-40 g de proteína de alta qualidade                  │
│  • Razão: "Resistência anabólica" com envelhecimento                   │
│                                                                         │
│  FÓRMULA:                                                              │
│  Leucina necessária = 0.04 g/kg de peso corporal por refeição          │
│  (ou ~0.18 g/lb)                                                       │
│                                                                         │
│  EXEMPLO (80 kg):                                                      │
│  80 × 0.04 = 3.2 g leucina por refeição                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.3 Conteúdo de Leucina por Fonte Proteica

| Fonte | Leucina (% da proteína) | Para 3g leucina |
|-------|------------------------|-----------------|
| Whey protein | 10-12% | ~25-30g proteína |
| Caseína | 8-9% | ~33-38g proteína |
| Ovos | 8.6% | ~35g proteína |
| Carne bovina | 8% | ~37-40g proteína |
| Frango | 7.5% | ~40g proteína |
| Peixe | 7-8% | ~38-43g proteína |
| Soja | 7.5% | ~40g proteína |
| Arroz + Feijão | 6-7% | ~45-50g proteína |
| Leguminosas | 6% | ~50g proteína |

### 3.4 Implicações Práticas

```
REGRAS DO LEUCINE THRESHOLD:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  ✓ Cada refeição deve conter proteína suficiente para atingir         │
│    o threshold de leucina (~2.5-3g para jovens, ~3-4g para idosos)    │
│                                                                         │
│  ✓ Proteínas de alta qualidade (animais, whey) atingem o threshold    │
│    com menos quantidade total                                          │
│                                                                         │
│  ✓ Proteínas vegetais precisam de maior quantidade ou combinação      │
│    para atingir o threshold                                            │
│                                                                         │
│  ✗ Suplementar leucina isolada NÃO substitui proteína completa        │
│    (precisa de TODOS os aminoácidos essenciais como "building blocks")│
│                                                                         │
│  ✗ Refeições com <15g proteína dificilmente atingem o threshold       │
│    e não estimulam MPS de forma ótima                                  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 4. PROTEÍNA POR REFEIÇÃO

### 4.1 O Mito do "30g por Refeição"

```
MITO: "O corpo só absorve 30g de proteína por refeição"

REALIDADE:
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  • O corpo ABSORVE praticamente toda proteína ingerida                 │
│    (digestão é muito eficiente)                                        │
│                                                                         │
│  • O que tem LIMITE é a estimulação de MPS por refeição:               │
│    - MPS maximizada com ~20-40g (dependendo do contexto)               │
│    - Acima disso: aminoácidos são oxidados ou usados em                │
│      outras funções (não "desperdiçados")                              │
│                                                                         │
│  • Proteína "extra" ainda contribui para:                              │
│    - Supressão de breakdown muscular (MPB)                             │
│    - Síntese de outras proteínas corporais                             │
│    - Gliconeogênese (em déficit)                                       │
│    - Efeito térmico dos alimentos                                      │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4.2 Quantidade Ideal por Refeição

Baseado em Schoenfeld & Aragon (2018):

```
PROTEÍNA POR REFEIÇÃO PARA MAXIMIZAR MPS:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  FÓRMULA:                                                              │
│  • 0.4 g/kg de peso corporal por refeição                              │
│  • (Máximo ~0.55 g/kg considerando variação individual)                │
│                                                                         │
│  EXEMPLOS:                                                             │
│  ────────────────────────────────────────────────────────────────────  │
│  Peso (kg)    Proteína/refeição (g)    Range (g)                       │
│  ────────────────────────────────────────────────────────────────────  │
│  60           24                        24-33                          │
│  70           28                        28-39                          │
│  80           32                        32-44                          │
│  90           36                        36-50                          │
│  100          40                        40-55                          │
│                                                                         │
│  REGRA PRÁTICA:                                                        │
│  • Jovens: 25-40g por refeição                                         │
│  • Idosos: 35-50g por refeição (resistência anabólica)                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4.3 Pesquisa sobre Doses Maiores

Estudo de Macnaughton et al. (2016):

```
COMPARAÇÃO: 20g vs 40g de whey após treino de corpo inteiro

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  Resultado: 40g produziu MPS ~20% MAIOR que 20g                        │
│                                                                         │
│  Contexto importante:                                                  │
│  • Treino de CORPO INTEIRO (não apenas pernas)                         │
│  • Maior massa muscular exercitada = maior demanda                     │
│                                                                         │
│  Implicação:                                                           │
│  • Após treinos de corpo inteiro: 40g pode ser melhor                  │
│  • Após treinos isolados (só braço): 20-25g é suficiente              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 5.2 Frequência de Refeições Proteicas

```
NÚMERO DE REFEIÇÕES COM PROTEÍNA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  MÍNIMO: 3 refeições/dia                                               │
│  • Cada uma atingindo leucine threshold                                │
│  • Espaçadas ~4-5 horas                                                │
│                                                                         │
│  IDEAL: 4 refeições/dia                                                │
│  • Cada uma com 25-40g proteína                                        │
│  • Espaçadas ~3-4 horas                                                │
│  • 4 picos de MPS por dia                                              │
│                                                                         │
│  5+ REFEIÇÕES: Retornos diminuídos                                     │
│  • MPS tem período refratário (~2-3h)                                  │
│  • Mais refeições não significam mais MPS                              │
│  • Pode dificultar atingir threshold por refeição                      │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 5.3 Período Refratário da MPS

```
MUSCLE FULL EFFECT:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  • Após atingir leucine threshold, MPS fica elevada por ~2-3 horas    │
│  • Depois disso, MPS RETORNA AO BASELINE                               │
│  • Mesmo que leucina ainda esteja elevada no sangue                    │
│  • Músculo fica "refratário" a novo estímulo por ~1-2 horas            │
│                                                                         │
│  IMPLICAÇÃO:                                                           │
│  • Não adianta comer proteína a cada 1-2 horas                         │
│  • Espaçar refeições 3-5 horas permite novo pico de MPS                │
│  • Grazing (comer pouquinho o tempo todo) é MENOS eficiente            │
│                                                                         │
│  TIMELINE:                                                             │
│  ─────────────────────────────────────────────────────────────         │
│  Hora 0: Refeição com proteína → MPS aumenta                           │
│  Hora 2-3: MPS volta ao baseline                                       │
│  Hora 3-4: Período refratário termina                                  │
│  Hora 4-5: Nova refeição pode estimular MPS novamente                  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 6. TIMING DE PROTEÍNA

### 6.1 Janela Anabólica (Anabolic Window)

```
O MITO DA "JANELA DE 30 MINUTOS":

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  MITO: "Você PRECISA consumir proteína em 30 minutos pós-treino        │
│         ou perde os ganhos"                                            │
│                                                                         │
│  REALIDADE (meta-análise Schoenfeld et al., 2013):                     │
│  • A "janela anabólica" é muito mais ampla (~4-6 horas)                │
│  • Total diário de proteína é MAIS IMPORTANTE que timing              │
│  • Timing tem efeito pequeno quando proteína total é adequada          │
│                                                                         │
│  QUANDO TIMING IMPORTA MAIS:                                           │
│  • Treino em jejum (nenhuma refeição nas últimas 4-6h)                 │
│  • Múltiplos treinos por dia                                           │
│  • Atletas de elite otimizando cada detalhe                            │
│                                                                         │
│  QUANDO TIMING IMPORTA MENOS:                                          │
│  • Se comeu refeição proteica 2-3h antes do treino                     │
│  • Se vai comer refeição proteica 2-3h após o treino                   │
│  • Para a maioria dos praticantes recreativos                          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 6.2 Recomendações de Timing

```
TIMING PRÁTICO DE PROTEÍNA:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  PRÉ-TREINO (1-3 horas antes):                                         │
│  • 25-40g proteína na refeição                                         │
│  • Aminoácidos estarão disponíveis durante e após treino               │
│  • Se treinar em jejum: considerar 10-20g whey 30min antes             │
│                                                                         │
│  PÓS-TREINO (até 2-3 horas depois):                                    │
│  • 25-40g proteína                                                     │
│  • Whey é conveniente mas não obrigatório                              │
│  • Refeição completa funciona igualmente bem                           │
│                                                                         │
│  ANTES DE DORMIR:                                                      │
│  • 30-40g proteína (preferencialmente caseína ou cottage)              │
│  • Digestão lenta = aminoácidos durante a noite                        │
│  • Especialmente importante para idosos                                │
│                                                                         │
│  AO ACORDAR:                                                           │
│  • 25-40g proteína                                                     │
│  • Interrompe jejum noturno                                            │
│  • Inicia primeiro pico de MPS do dia                                  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 7. PROTEÍNA EM DÉFICIT CALÓRICO

### 7.1 Por que Aumentar Proteína no Cutting?

```
RAZÕES PARA MAIS PROTEÍNA EM DÉFICIT:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  1. PRESERVAR MASSA MUSCULAR                                           │
│     • Déficit aumenta catabolismo (breakdown)                          │
│     • Mais proteína = mais substrato para reconstrução                 │
│                                                                         │
│  2. MAIOR OXIDAÇÃO DE AMINOÁCIDOS                                      │
│     • Em déficit, corpo usa mais proteína para energia                 │
│     • Menos aminoácidos disponíveis para MPS                           │
│     • Compensar com maior ingestão                                     │
│                                                                         │
│  3. SACIEDADE                                                          │
│     • Proteína é o macronutriente mais saciante                        │
│     • Ajuda a controlar fome em déficit                                │
│                                                                         │
│  4. EFEITO TÉRMICO                                                     │
│     • TEF da proteína: 20-35%                                          │
│     • Maior gasto calórico na digestão                                 │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 7.2 Recomendações em Déficit

Baseado em Helms et al. (2014) e pesquisas subsequentes:

```
PROTEÍNA EM DÉFICIT CALÓRICO:

┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│  DÉFICIT MODERADO (-300 a -500 kcal):                                  │
│  • 2.0-2.4 g/kg/dia                                                    │
│                                                                         │
│  DÉFICIT AGRESSIVO (-500 a -1000 kcal):                                │
│  • 2.4-3.1 g/kg/dia                                                    │
│                                                                         │
│  FÓRMULA POR MASSA MAGRA:                                              │
│  • 2.3-3.1 g/kg de MASSA MAGRA (não peso total)                        │
│                                                                         │
│  EXEMPLO (80kg, 15% BF):                                               │
│  ─────────────────────────────────────────────────────────────         │
│  Massa magra = 80 × 0.85 = 68 kg                                       │
│  Proteína mínima = 68 × 2.3 = 156g/dia                                 │
│  Proteína ideal = 68 × 2.8 = 190g/dia                                  │
│  Ou simplesmente: 80 × 2.2 = 176g/dia (pelo peso total)                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 9. COMO O VITRÚVIO DEVE CALCULAR

### 9.1 Algoritmo de Prescrição

```typescript
interface ParametrosProteina {
  peso: number              // kg
  gorduraCorporal: number   // %
  idade: number
  objetivo: 'BULK' | 'MANUTENCAO' | 'CUT'
  deficitCalorico?: number  // kcal
  nivelTreino: 'INICIANTE' | 'INTERMEDIARIO' | 'AVANCADO'
  dietaBase: 'ONIVORO' | 'VEGETARIANO' | 'VEGANO'
}

interface RecomendacaoProteina {
  proteinaDiaria: number       // g/dia
  proteinaKg: number           // g/kg/dia
  proteinaPorRefeicao: number  // g/refeição
  numeroRefeicoes: number
  leucinaPorRefeicao: number   // g
  fontesPrioritarias: string[]
  notas: string[]
}

function calcularProteina(params: ParametrosProteina): RecomendacaoProteina {
  const massaMagra = params.peso * (1 - params.gorduraCorporal / 100)
  
  // Base por objetivo
  let proteinaKgBase: number
  switch (params.objetivo) {
    case 'BULK':
      proteinaKgBase = 2.0  // g/kg
      break
    case 'MANUTENCAO':
      proteinaKgBase = 1.8
      break
    case 'CUT':
      // Escalar com agressividade do déficit
      if (params.deficitCalorico && params.deficitCalorico > 500) {
        proteinaKgBase = 2.6
      } else {
        proteinaKgBase = 2.2
      }
      break
  }
  
  // Ajuste por idade
  if (params.idade > 50) {
    proteinaKgBase += 0.2
  } else if (params.idade > 60) {
    proteinaKgBase += 0.3
  }
  
  // Ajuste por nível de treino
  if (params.nivelTreino === 'AVANCADO') {
    proteinaKgBase += 0.1  // maior turnover
  }
  
  // Ajuste para dieta vegetal
  if (params.dietaBase === 'VEGANO') {
    proteinaKgBase *= 1.2  // +20% para compensar biodisponibilidade
  } else if (params.dietaBase === 'VEGETARIANO') {
    proteinaKgBase *= 1.1  // +10%
  }
  
  // Calcular total diário
  // Para obesos, usar peso ajustado
  let pesoCalculo = params.peso
  if (params.gorduraCorporal > 25) {
    pesoCalculo = massaMagra + (params.peso - massaMagra) * 0.25
  }
  
  const proteinaDiaria = Math.round(pesoCalculo * proteinaKgBase)
  
  // Distribuição em refeições
  const numeroRefeicoes = 4  // ideal
  const proteinaPorRefeicao = Math.round(proteinaDiaria / numeroRefeicoes)
  
  // Leucina por refeição
  let leucinaPorRefeicao = 2.5
  if (params.idade > 50) leucinaPorRefeicao = 3.0
  if (params.idade > 60) leucinaPorRefeicao = 3.5
  
  // Fontes prioritárias
  const fontesPrioritarias = params.dietaBase === 'VEGANO' 
    ? ['Proteína de ervilha + arroz', 'Tofu', 'Tempeh', 'Seitan', 'Leguminosas']
    : params.dietaBase === 'VEGETARIANO'
    ? ['Ovos', 'Whey', 'Iogurte grego', 'Cottage', 'Leguminosas']
    : ['Frango', 'Peixe', 'Ovos', 'Carne bovina', 'Whey', 'Iogurte grego']
  
  // Notas específicas
  const notas: string[] = []
  if (params.objetivo === 'CUT') {
    notas.push('Proteína elevada para preservar massa muscular em déficit')
  }
  if (params.idade > 50) {
    notas.push('Quantidade aumentada devido à resistência anabólica da idade')
  }
  if (params.dietaBase !== 'ONIVORO') {
    notas.push('Combine fontes vegetais para perfil completo de aminoácidos')
  }
  if (proteinaPorRefeicao > 50) {
    notas.push('Considere adicionar 5ª refeição para melhor distribuição')
  }
  
  return {
    proteinaDiaria,
    proteinaKg: proteinaKgBase,
    proteinaPorRefeicao,
    numeroRefeicoes,
    leucinaPorRefeicao,
    fontesPrioritarias,
    notas
  }
}
```

### 10.2 Regras de Ouro

```
REGRAS DE PROTEÍNA DO VITRÚVIO:

1. TOTAL DIÁRIO É PRIORIDADE #1
   • Mais importante que timing ou distribuição
   • Mínimo 1.6 g/kg, ideal 2.0 g/kg para hipertrofia

2. CADA REFEIÇÃO DEVE ATINGIR LEUCINE THRESHOLD
   • Mínimo 25g proteína de qualidade por refeição
   • Ou 2.5-3g de leucina

3. DISTRIBUIR EM 3-4 REFEIÇÕES
   • Espaçadas 3-5 horas
   • Evitar concentrar tudo em 1-2 refeições

4. AUMENTAR EM DÉFICIT CALÓRICO
   • +0.4-0.6 g/kg adicional durante cutting
   • Preserva massa muscular

5. AJUSTAR POR IDADE
   • >50 anos: aumentar 0.2 g/kg
   • >60 anos: aumentar 0.3 g/kg
   • Proteína antes de dormir é especialmente importante

6. QUALIDADE IMPORTA
   • Priorizar fontes completas (animais, soja)
   • Veganos: combinar fontes e aumentar quantidade
```
